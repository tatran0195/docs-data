## Description

Full Circle Vertex Adjust

## Syntax

```psj
GeomEditFCircVertexAdjust(Cursor Body)
```

## Inputs

### `1. Cursor`

Target Body for Vertex Adjust

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
GeomEditFCircVertexAdjust(3:2)
```
