## Description

Full Circle Vertex Adjust

## Syntax

```psj
MeshEditAdjustVertex(cursor[] taBodyCur)
```

## Inputs

### `1. Cursor[]`

Part cursor([3:Part ID])

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
MeshEditAdjustVertex([3:1])
```
