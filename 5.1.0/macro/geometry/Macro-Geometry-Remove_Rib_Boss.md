## Description

Remove Rib Boss

## Syntax

```psj
Remove_Rib_Boss(Curor[] Face, double Gradation, int iContinuity)
```

## Inputs

### `1. Cursor[]`

Face cursor([6:Face ID])

### `2. Double`

Gradation value

### `3. Int`

Connection Continuity: 0,1,2

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
Remove_Rib_Boss([624], 1, 1)
```
