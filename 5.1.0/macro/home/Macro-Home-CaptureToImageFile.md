## Description

Capture to Image File.

## Syntax

```psj
CaptureToImageFile(string FilePath)
```

## Inputs

### `1. String`

Image file path(extension = \*.bmp or \*.jpg or \*.gif or \*.png)

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
CaptureToImageFile("D:/Test.jpg")
```
