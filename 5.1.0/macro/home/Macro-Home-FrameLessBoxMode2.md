## Description

Display opening documents in vertical and horizontal arrangement like a box, in mode 2.

## Syntax

```psj
FrameLessBoxMode2()
```

## Inputs

The macro has no input.

## Return Code

No return value.

## Sample Code

```psj
FrameLessBoxMode2()
```
