## Description

Display documents in a horizontal arrangement in Mode 1.

## Syntax

```psj
FrameLessHorizontalMode1()
```

## Inputs

No input.

## Return Code

No return value.

## Sample Code

```psj
FrameLessHorizontalMode1()
```
