## Description

Display documents in a horizontal arrangement in Mode 2.

## Syntax

```psj
FrameLessHorizontalMode2()
```

## Inputs

No input.

## Return Code

No return value.

## Sample Code

```psj
FrameLessHorizontalMode2()
```
