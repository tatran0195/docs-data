## Description

Display documents in a horizontal arrangement in Mode 3.

## Syntax

```psj
FrameLessHorizontalMode3()
```

## Inputs

No input.

## Return Code

No return value.

## Sample Code

```psj
FrameLessHorizontalMode3()
```
