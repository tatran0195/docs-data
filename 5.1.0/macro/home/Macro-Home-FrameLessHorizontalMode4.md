## Description

Display documents in a horizontal arrangement in Mode 4.

## Syntax

```psj
FrameLessHorizontalMode4()
```

## Inputs

No input.

## Return Code

No return value.

## Sample Code

```psj
FrameLessHorizontalMode4()
```
