## Description

Display documents in a horizontal arrangement in Mode 5.

## Syntax

```psj
FrameLessHorizontalMode5()
```

## Inputs

No input.

## Return Code

No return value.

## Sample Code

```psj
FrameLessHorizontalMode5()
```
