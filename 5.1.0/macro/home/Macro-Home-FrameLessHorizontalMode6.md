## Description

Display documents in a horizontal arrangement in Mode 6.

## Syntax

```psj
FrameLessHorizontalMode6()
```

## Inputs

No input.

## Return Code

No return value.

## Sample Code

```psj
FrameLessHorizontalMode6()
```
