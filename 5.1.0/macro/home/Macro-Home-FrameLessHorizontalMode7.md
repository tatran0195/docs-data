## Description

Display documents in a horizontal arrangement in Mode 7.

## Syntax

```psj
FrameLessHorizontalMode7()
```

## Inputs

No input.

## Return Code

No return value.

## Sample Code

```psj
FrameLessHorizontalMode7()
```
