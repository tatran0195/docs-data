## Description

Display documents in a vertical side-by-side arrangement in Mode 1.

## Syntax

```psj
FrameLessVerticalMode1()
```

## Inputs

No input.

## Return Code

No return value.

## Sample Code

```psj
FrameLessVerticalMode1()
```
