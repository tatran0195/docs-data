## Description

Display documents in a vertical side-by-side arrangement in Mode 2.

## Syntax

```psj
FrameLessVerticalMode2()
```

## Inputs

No input.

## Return Code

No return value.

## Sample Code

```psj
FrameLessVerticalMode2()
```
