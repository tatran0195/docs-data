## Description

Display documents in a vertical side-by-side arrangement in Mode 3.

## Syntax

```psj
FrameLessVerticalMode3()
```

## Inputs

No input.

## Return Code

No return value.

## Sample Code

```psj
FrameLessVerticalMode3()
```
