## Description

Display documents in a vertical side-by-side arrangement in Mode 4.

## Syntax

```psj
FrameLessVerticalMode4()
```

## Inputs

No input.

## Return Code

No return value.

## Sample Code

```psj
FrameLessVerticalMode4()
```
