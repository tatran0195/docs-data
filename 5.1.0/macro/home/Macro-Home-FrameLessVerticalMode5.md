## Description

Display documents in a vertical side-by-side arrangement in Mode 5.

## Syntax

```psj
FrameLessVerticalMode5()
```

## Inputs

No input.

## Return Code

No return value.

## Sample Code

```psj
FrameLessVerticalMode5()
```
