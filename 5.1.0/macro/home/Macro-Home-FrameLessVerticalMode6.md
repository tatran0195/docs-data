## Description

Display documents in a vertical side-by-side arrangement in Mode 6.

## Syntax

```psj
FrameLessVerticalMode6()
```

## Inputs

No input.

## Return Code

No return value.

## Sample Code

```psj
FrameLessVerticalMode6()
```
