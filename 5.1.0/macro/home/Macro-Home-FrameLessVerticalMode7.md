## Description

Display documents in a vertical side-by-side arrangement in Mode 7.

## Syntax

```psj
FrameLessVerticalMode7()
```

## Inputs

No input.

## Return Code

No return value.

## Sample Code

```psj
FrameLessVerticalMode7()
```
