## Description

Enable the specified user frame as current.

## Syntax

```psj
SelectUserViewFrame()
```

## Inputs

### `1. string`

Name of the user frame.

## Return Code

No return value.

## Sample Code

```psj
SelectUserViewFrame("New_Frame_1 ((Multiple))")
```
