## Description

Toggle the display of hidden parts or faces on the current document.

## Syntax

```psj
ShowHideFlip(int Type)
```

## Inputs

### `1. Int`

An Integer specifying the target type to toggle the displaying. 

## Return Code

- "1": The function can be executed.
- "0": The function cannot be executed.

## Sample Code

```psj
ShowHideFlip(0)
```
