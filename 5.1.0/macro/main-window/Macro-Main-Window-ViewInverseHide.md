## Description

Invert hide targets by context menu

## Syntax

```psj
ViewInverseHide(cursor[] Targets)
```

## Inputs

### `1. Cursor[]`

A Cursor List specifying the targets to be inverted hide. 

## Return Code

- "1": The function can be executed.
- "0": The function cannot be executed.

## Sample Code

```psj
ViewInverseHide([])
```
