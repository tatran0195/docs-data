## Description

Select all faces or shell elements according to the specified target type.

## Syntax

```psj
ViewSelectAll(int TargetType)
```

## Inputs

### `1. Int`

An Integer specifying the target type to be selected.

## Return Code

A Cursor List specifying the selected targets.

## Sample Code

```psj
ViewSelectAll(3)
```
