## Description

Show all hidden entities.

## Syntax

```psj
ViewShowAll()
```

## Inputs

Nothing.

## Return Code

- "1": The function can be executed.
- "0": The function cannot be executed.

## Sample Code

```psj
ViewShowAll()
```
