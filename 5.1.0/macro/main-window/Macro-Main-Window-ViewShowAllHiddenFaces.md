## Description

Show all hidden faces of displaying bodies.

## Syntax

```psj
ViewShowAllHiddenFaces()
```

## Inputs

Nothing.

## Return Code

- "1": The function can be executed.
- "0": The function cannot be executed.

## Sample Code

```psj
ViewShowAllHiddenFaces()
```
