## Description

Swap Element Edge

## Syntax

```psj
SwapElemEdge(cursor[] vcrElemEdge)
```

## Inputs

### `1. Cursor[]`

Select (multiple) element edges cursor([10:Node ID-10:Node ID])

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
SwapElemEdge([10:82-10:117, 10:81-10:116])
```
