## Description

Mirror Copy

## Syntax

```psj
MirrorCopy(cursor[] taFace)
```

## Inputs

### `1. Cursor[]`

Target face cursor([6:Face ID])

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
MirrorCopy([6:21, 6:57])
```
