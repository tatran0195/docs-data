## Description

Load local setting

## Syntax

```psj
ImportMeshSetting(string fname)
```

## Inputs

### `1. String`

load local setting from this file

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
ImportMeshSetting("D:/test.xml")
```
