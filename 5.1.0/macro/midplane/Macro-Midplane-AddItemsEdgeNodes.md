## Description

Add Items Edge from Nodes

## Syntax

```psj
AddItemsEdgeNodes(int[] NodeKey)
```

## Inputs

### `1. Int[]`

Target Nodes for Creating New Edge

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
AddItemsEdgeNodes([180, 188, 196, 204])
```
