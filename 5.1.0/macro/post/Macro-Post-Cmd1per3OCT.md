## Description

1/3_OCT turn ON/OFF

## Syntax

```psj
Cmd1per3OCT(int flag)
```

## Inputs

### `1. int `

1/3_OCT turn ON/OFF. 0: OFF, 1: ON.

## Return Code

Nothing.

## Sample Code

```psj
Cmd1per3OCT(1)
```
