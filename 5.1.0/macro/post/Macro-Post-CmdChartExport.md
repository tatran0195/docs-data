## Description

Save content of chart view in .csv file.

## Syntax

```psj
CmdChartExport(string file)
```

## Inputs

### `1. string `

Save file path.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartExport("C:/temp/file.csv")
```
