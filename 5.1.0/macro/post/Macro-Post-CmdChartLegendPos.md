## Description

Set legend position of chart.

## Syntax

```psj
CmdChartLegendPos(int position)
```

## Inputs

### `1. int `

Set legend position. 1: Top, 2: Bottom, 3: Left, 4: Right.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartLegendPos(0)
```
