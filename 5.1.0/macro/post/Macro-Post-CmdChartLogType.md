## Description

Set log type.

## Syntax

```psj
CmdChartLogType(int type)
```

## Inputs

### `1. int `

Log Type. 0:General, 1:Log, 2:10log(10), 3:dB, 4:SPL.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartLogType(0)
```
