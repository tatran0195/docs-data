## Description

Pan chart view.

## Syntax

```psj
CmdChartPan(float distX, float distY)
```

## Inputs

### `1. float `

X offset value.

### `2. float `

Y offset value.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartPan(141.000000,-42.000000)
```
