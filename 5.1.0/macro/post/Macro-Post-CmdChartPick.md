## Description

Enable / disable Pick Data.

## Syntax

```psj
CmdChartPick(int PickData)
```

## Inputs

### `1. int `

Enable / disable Pick Data. 1: Enable, 0: Disable.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartPick(1)
```
