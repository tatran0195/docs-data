## Description

Set Hovered point.

## Syntax

```psj
CmdChartSetPointHighlight(color col, int size)
```

## Inputs

### `1. color `

Highlighted Data Point Color.

### `2. int `

Highlighted Data Point Size.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartSetPointHighlight(65280, 12)
```
