## Description

Set chart range by manual.

## Syntax

```psj
CmdChartSetRange(double MinX, double MaxX, double IntervalX, double MinY, double MaxY, double IntervalY)
```

## Inputs

### `1. double `

Minimum X-Axis

### `2. double `

Maximum X-Axis

### `3. double `

Tick Interval X-Axis

### `4. double `

Minimum Y-Axis

### `5. double `

Maximum Y-Axis

### `6. double `

Tick Interval Y-Axis

## Return Code

Nothing.

## Sample Code

```psj
CmdChartSetRange(0,540,20,-30,20,5)
```
