## Description

Set title of X axis.

## Syntax

```psj
CmdChartSetTitleX(string title, int pos)
```

## Inputs

### `1. string `

Title string.

### `2. int `

Indicate title position.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartSetTitleX(My Title,0)
```
