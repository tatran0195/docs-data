## Description

Set title of Y axis.

## Syntax

```psj
CmdChartSetTitleY(string title)
```

## Inputs

### `1. string `

Title string.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartSetTitleY("My Title")
```
