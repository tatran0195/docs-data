## Description

Show / Hide data label.

## Syntax

```psj
CmdChartShowDataLabel(int flag)
```

## Inputs

### `1. int `

Show / Hide data label. 0: Hide, 1: Show.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartShowDataLabel(1)
```
