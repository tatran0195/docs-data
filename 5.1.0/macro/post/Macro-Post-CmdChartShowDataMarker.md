## Description

Show / Hide data marker.

## Syntax

```psj
CmdChartShowDataMarker(int flag)
```

## Inputs

### `1. int `

Show / Hide data maker. 0: Hide, 1: Show.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartShowDataMarker(1)
```
