## Description

Show / Hide info bar.

## Syntax

```psj
CmdChartShowInfoBar(int flag)
```

## Inputs

### `1. int `

Show / Hide info bar. 0: Hide, 1: Show.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartShowInfoBar(1)
```
