## Description

Show / Hide legend.

## Syntax

```psj
CmdChartShowLegend(int flag)
```

## Inputs

### `1. int `

Show / Hide legend. 0: Hide, 1: Show.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartShowLegend(1)
```
