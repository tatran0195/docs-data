## Description

Show / Hide title.

## Syntax

```psj
CmdChartShowTitle(int flag)
```

## Inputs

### `1. int `

Show / Hide title. 0: Hide, 1: Show.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartShowTitle(1)
```
