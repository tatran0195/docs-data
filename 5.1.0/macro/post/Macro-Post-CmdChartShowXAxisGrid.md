## Description

Show / Hide X-Axis Grid.

## Syntax

```psj
CmdChartShowXAxisGrid(int flag)
```

## Inputs

### `1. int `

Show / Hide X-Axis Grid. 0: Hide, 1: Show.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartShowXAxisGrid(1)
```
