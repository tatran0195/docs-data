## Description

Show / Hide X-Axis Interval.

## Syntax

```psj
CmdChartShowXAxisInterval(int flag)
```

## Inputs

### `1. int `

Show / Hide X-Axis Interval. 0: Hide, 1: Show.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartShowXAxisInterval(1)
```
