## Description

Show / Hide X-Axis Name.

## Syntax

```psj
CmdChartShowXAxisName(int flag)
```

## Inputs

### `1. int `

Show / Hide  X-Axis Name 0: Hide, 1: Show.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartShowXAxisName(1)
```
