## Description

Show / Hide Data label as X,Y.

## Syntax

```psj
CmdChartShowXYLabel(int flag)
```

## Inputs

### `1. int `

Show / Hide Data label as X,Y. 0: Hide, 1: Show.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartShowXYLabel(1)
```
