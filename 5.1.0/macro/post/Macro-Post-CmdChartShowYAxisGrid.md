## Description

Show / Hide Y-Axis Grid.

## Syntax

```psj
CmdChartShowYAxisGrid(int flag)
```

## Inputs

### `1. int `

Show / Hide Y-Axis Grid. 0: Hide, 1: Show.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartShowYAxisGrid(1)
```
