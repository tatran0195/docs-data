## Description

Show / Hide Y-Axis Interval.

## Syntax

```psj
CmdChartShowYAxisInterval(int flag)
```

## Inputs

### `1. int `

Show / Hide Y-Axis Interval. 0: Hide, 1: Show.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartShowYAxisInterval(1)
```
