## Description

Show / Hide Y-Axis Name.

## Syntax

```psj
CmdChartShowYAxisName(int flag)
```

## Inputs

### `1. int `

Show / Hide  Y-Axis Name 0: Hide, 1: Show.

## Return Code

Nothing.

## Sample Code

```psj
CmdChartShowYAxisName(1)
```
