## Description

Zoom chart view.

## Syntax

```psj
CCmdChartZoom(float pt1x, float pt1y, float pt2x, float pt2y)
```

## Inputs

### `1. float `

X1 point.

### `2. float `

Y1 point.

### `3. float `

X2 point

### `4. float `

Y2 point.

## Return Code

Nothing.

## Sample Code

```psj
CCmdChartZoom(503.000000,356.000000,441.000000,376.000000)
```
