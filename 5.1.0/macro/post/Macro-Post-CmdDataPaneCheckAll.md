## Description

Turn all the check boxes of the data in current tab of Watch Data Window on.

## Syntax

```psj
CmdDataPaneCheckAll()
```

## Inputs

Nothing.

## Return Code

Nothing.

## Sample Code

```psj
CmdDataPaneCheckAll()
```
