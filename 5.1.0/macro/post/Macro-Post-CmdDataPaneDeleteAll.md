## Description

Delete all the data in current tab of Watch Data Window.

## Syntax

```psj
CmdDataPaneDeleteAll()
```

## Inputs

Nothing.

## Return Code

Nothing.

## Sample Code

```psj
CmdDataPaneDeleteAll()
```
