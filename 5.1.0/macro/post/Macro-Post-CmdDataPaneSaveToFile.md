## Description

Save current tab of Watch Data Window into csv file.

## Syntax

```psj
CmdDataPaneSaveToFile(string file)
```

## Inputs

### `1. string `

Save file path.

## Return Code

Nothing.

## Sample Code

```psj
CmdDataPaneSaveToFile("C:/Temp/Data.csv")
```
