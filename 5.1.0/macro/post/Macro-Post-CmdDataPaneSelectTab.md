## Description

Activate specific tab of Watch Window 

## Syntax

```psj
CmdDataPaneSelectTab(int TabID)
```

## Inputs

### `1. int`

Tab ID. 
1 : Node
2 : Element
3 : Position
4 : Node Plot Scan
5 : Element Plot Scan
6 : Position Plot Scan
7 : Strain Gauge
8 : PeakSearch
9 : Max/Min
10 : Area Max/Min
11 : Fatigue Strength
12 : Compare 

## Return Code

- Nothing.

## Sample Code

```psj
CmdDataPaneSelectTab(3)
```
