## Description

Turn all the check boxes of the data in current tab of Watch Data Window off.

## Syntax

```psj
CmdDataPaneUncheckAll()
```

## Inputs

Nothing.

## Return Code

Nothing.

## Sample Code

```psj
CmdDataPaneUncheckAll()
```
