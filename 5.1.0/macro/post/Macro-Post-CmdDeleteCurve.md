## Description

Delete curve.

## Syntax

```psj
CmdDeleteCurve(int[] curveId)
```

## Inputs

### `1. int [] `

List of curve id.

## Return Code

Nothing.

## Sample Code

```psj
CmdDeleteCurve([1, 2])
```
