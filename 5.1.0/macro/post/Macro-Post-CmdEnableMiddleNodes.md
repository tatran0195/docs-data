## Description

Enable / disable middle node

## Syntax

```psj
CmdEnableMiddleNodes(int Flag)
```

## Inputs

### `1. int`

Enable / disable middle node.
0 : disable
1 : enable

## Return Code

Nothing.

## Sample Code

```psj
CmdEnableMiddleNodes(1)
```
