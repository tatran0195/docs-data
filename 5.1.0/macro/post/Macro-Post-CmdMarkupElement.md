## Description

Add note to selected Elements.

## Syntax

```psj
 CmdMarkupElement(int[] Elements)
```

## Inputs


### `1. int[]  `

Element IDs to add note.


## Return Code

Nothing.

## Sample Code

```psj
CmdMarkupElement([2974,3000,3006])
```
