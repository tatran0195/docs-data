## Description

Add note to selected node.

## Syntax

```psj
 CmdMarkupNode(int[] nodes)
```
## Inputs
### `1. int[]  `

Node IDs to add note.

## Return Code

Nothing.

## Sample Code

```psj
CmdMarkupNode([4,978])
```
