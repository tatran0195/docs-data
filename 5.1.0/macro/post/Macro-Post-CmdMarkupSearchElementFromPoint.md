## Description

Search nearest element to indicated position and put note at the element.

## Syntax

```psj
CmdMarkupSearchNodesFromMultiPoints(double x, double y, double z)
```

## Inputs

### `1. doulbe `

X position of search point.

### `2. doulbe `

Y position of search point.

### `3. doulbe `

Z position of search point.

## Return Code

### `1. bool `

Succeed(1) or failed(0).

## Sample Code

```psj
CmdMarkupSearchElementFromPoint(0.000000, 0.000000, 0.000000)
>> 1
```
