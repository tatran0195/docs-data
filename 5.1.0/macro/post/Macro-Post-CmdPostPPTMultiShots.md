## Description

Create ppt of modal analysis.

## Syntax

```psj
CmdPostPPTMultiShots(double startFreq, double endFreq)
```

## Inputs

### `1. double `

Start Frequency.

### `2.double `

End Frequency.

## Return Code

Nothing.

## Sample Code

```psj
CmdPostPPTMultiShots(0.000000, 1000.000000)
```
