## Description

Display or hide Max Min label.

## Syntax

```psj
CmdShowHideMaxMin(string Flag)
```

## Inputs

### `1. string`

[TRUE] or [FALSE].
[TRUE]: Show Max Min label.
[FALSE]: Hide Max Min label.

## Return Code

Nothing.

## Sample Code

```psj
CmdShowHideMaxMin([TRUE])
```
