## Description

Copy specified template.

## Syntax

```psj
CopyTemplate(string CurrentName, string NewTemplate)
```

## Inputs

### `1. string `

Copy source template's name.

### `2. string `

Copied templates' name.

## Return Code

- "1": Succeeded.
- "0": Failed.

## Sample Code

```psj
CopyTemplate("Current Template","New Template")
```
