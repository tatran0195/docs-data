## Description

Create new chart window for current post document.

## Syntax

```psj
CreateNewChartWindow()
```

## Inputs

No arguments

## Return Code

- "1": If succeeded.
- "FALSE": If failed.

## Sample Code

```psj
CreateNewChartWindow()
```
