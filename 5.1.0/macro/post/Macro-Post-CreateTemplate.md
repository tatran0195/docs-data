## Description

Create a new template.

## Syntax

```psj
CreateTemplate(string templateName, string comment)
```

## Inputs

### `1. string `

Template name.

### `2. string `

Comment for the template.

## Return Code

- "1": Succeeded.
- "0": Failed.

## Sample Code

```psj
CreateTemplate("New Template", "For my default template.")
```
