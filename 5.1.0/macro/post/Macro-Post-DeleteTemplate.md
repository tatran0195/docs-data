## Description

Delete specified template from document.

## Syntax

```psj
DeleteTemplate(string TemplateName)
```

## Inputs

### `1. string `

Template name.

## Return Code

- "1": Succeeded.
- "0": Failed.

## Sample Code

```psj
DeleteTemplate("My Template")
```
