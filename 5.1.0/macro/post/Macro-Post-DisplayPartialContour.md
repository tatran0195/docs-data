## Description

Show contour partially

## Syntax

```psj
DisplayPartialContour(int flag, Cursor[] entities)
```

## Inputs

### `1. int  `

Use Part Color flag

### `2. cursor[]  `

List of target entity

## Return Code

Nothing.

## Sample Code

```psj
DisplayPartialContour(1, [6:10, 6:9])

```
