## Description

Enable / Disable Optimization Render Mode. 
This mode can display hidden elements set by Optimized Shape parameter on Optimization Toolkit. 

## Syntax

```psj
EnableOptimizationRenderMode(int Flag)
```

## Inputs

### `1. int `

Turn on/off optimization render mode flag. 0: Off, 1: On.

## Return Code

Nothing.

## Sample Code

```psj
EnableOptimizationRenderMode(1)
```
