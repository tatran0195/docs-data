## Description

Export templates to .xml file.

## Syntax

```psj
ExportTemplate(string file)
```

## Inputs

### `1. string `

Export file path.

## Return Code

- "1": Succeeded.
- "0": Failed.

## Sample Code

```psj
ExportTemplate("C:/Temp/Template.xml")
```
