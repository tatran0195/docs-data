## Description

Import templates from .xml file.

## Syntax

```psj
ImportTemplate(string file)
```

## Inputs

### `1. string `

Import file path.

## Return Code

- "1": Succeeded.
- "0": Failed.

## Sample Code

```psj
ImportTemplate("C:/Temp/Template.xml")
```
