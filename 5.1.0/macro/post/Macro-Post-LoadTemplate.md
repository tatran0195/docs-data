## Description

Load template to current document.

## Syntax

```psj
CreateTemplate(string templateName)
```

## Inputs

### `1. string `

Template name.

## Return Code

- "1": Succeeded.
- "0": Failed.

## Sample Code

```psj
LoadTemplate("My Template")
```
