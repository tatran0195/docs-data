## Description

Set the background color of notes.

## Syntax

```psj
SetNoteColorBackground(color bgColor)
```

## Inputs

### `1. color`

Specify the background color of notes.

## Return Code

Nothing.

## Sample Code

```psj
SetNoteColorBackground(9109643)
```
