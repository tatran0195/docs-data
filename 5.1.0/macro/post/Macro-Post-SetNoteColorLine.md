## Description

Set the color and width of outline of notes.

## Syntax

```psj
SetNoteColorLine(color outlineColor, int outlineWidth)
```

## Inputs

### `1. color`

Specify the outline color of notes.

### `2. int`

Specify the outline width of notes.

## Return Code

Nothing.

## Sample Code

```psj
SetNoteColorLine(8421376, 1)
```
