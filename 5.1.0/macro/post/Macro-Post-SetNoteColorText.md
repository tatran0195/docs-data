## Description

Set the color and size of text on notes.

## Syntax

```psj
SetNoteColorText(color textColor, int textSize)
```

## Inputs

### `1. color`

Specify the text color of notes.

### `2. int`

Specify the text size of notes.

## Return Code

Nothing.

## Sample Code

```psj
SetNoteColorText(9125192, 14)
```
