## Description

Sets the whether subcase ID to be displayed in the notes.

## Syntax

```psj
SetNoteResultSubID(bool diplay)
```

## Inputs

### `1. bool`

Specify whether display subcase ID on the notes or not.
- 0 : Do not display
- 1 : Display

## Return Code

Nothing.

## Sample Code

```psj
SetNoteResultSubID(1)
```
