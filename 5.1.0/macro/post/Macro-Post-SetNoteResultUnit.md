## Description

Sets the whether data unit to be displayed in the notes.

## Syntax

```psj
SetNoteResultUnit(bool diplay)
```

## Inputs

### `1. bool`

Specify whether display unit on the notes or not.
- 0 : Do not display
- 1 : Display

## Return Code

Nothing.

## Sample Code

```psj
SetNoteResultUnit(1)
```
