## Description

Show indicated contact pair data.

## Syntax

```psj
ShowContactPairData(string[] contactPairs)
```

## Inputs

### `1. string[] `

Contact pairs to show.

## Return Code

Nothing.

## Sample Code

```psj
ShowContactPairData(["c8", "c10"])
```
