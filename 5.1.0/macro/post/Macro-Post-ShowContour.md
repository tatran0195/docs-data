## Description

Show / Hide contour display.

## Syntax

```psj
ShowContour(int Show)
```

## Inputs

### `1. int `

Show contour flag. 0: Hide, 1: Show.

## Return Code

Nothing.

## Sample Code

```psj
ShowContour(1)
```
