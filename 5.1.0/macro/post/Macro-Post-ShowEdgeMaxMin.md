## Description

Show Max Min label for nodes in selected edges.

## Syntax

```psj
ShowEdgeMaxMin(cursor[] edges)
```

## Inputs

### `1. cursor `

Edges.

## Return Code

Nothing.

## Sample Code

```psj
ShowEdgeMaxMin([5:42, 5:43, 5:44, 5:45])
```
