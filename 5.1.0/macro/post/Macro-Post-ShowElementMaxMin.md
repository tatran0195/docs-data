## Description

Show Max Min label for selected elements.

## Syntax

```psj
ShowElementMaxMin(cursor[] elements)
```

## Inputs

### `1. cursor[] `

Element list.

## Return Code

Nothing.

## Sample Code

```psj
ShowElementMaxMin([193:2996, 193:2979])
```
