## Description

Show Max Min label for nodes in selected faces.

## Syntax

```psj
ShowFaceMaxMin(cursor[] faces)
```

## Inputs

### `1. cursor `

Faces.

## Return Code

Nothing.

## Sample Code

```psj
ShowFaceMaxMin([6:8, 6:6])
```
