## Description

Show / Hide contour legend

## Syntax

```psj
ShowLegend(int Show)
```

## Inputs

### `1. int `

Show legend flag. 0: Hide, 1: Show.

## Return Code

Nothing.

## Sample Code

```psj
ShowLegend(1)
```
