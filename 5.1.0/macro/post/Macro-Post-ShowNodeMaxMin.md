## Description

Show Max Min label for selected nodes.

## Syntax

```psj
ShowNodeMaxMin(cursor[] nodes)
```

## Inputs

### `1. cursor[] `

Node list.

## Return Code

Nothing.

## Sample Code

```psj
ShowNodeMaxMin([192:1032, 192:1030, 10:491, 10:989])
```
