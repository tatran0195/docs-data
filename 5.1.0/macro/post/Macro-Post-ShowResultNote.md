## Description

Delete all items in specific pane of Watch Data Window.

## Syntax

```psj
ShowResultNote(bool showFlag)
```

## Inputs

### `1. bool `

Show note flag. 0:Hide, 1:Show.

## Return Code

Nothing.

## Sample Code

```psj
ShowResultNote(1)
```
