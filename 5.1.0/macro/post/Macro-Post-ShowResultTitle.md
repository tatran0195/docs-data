## Description

Show / Hide result title.

## Syntax

```psj
ShowResultTitle(int Show)
```

## Inputs

### `1. int `

Show result title flag. 0: Hide, 1: Show.

## Return Code

Nothing.

## Sample Code

```psj
ShowResultTitle(1)
```
