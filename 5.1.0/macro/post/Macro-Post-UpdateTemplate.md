## Description

Load template to current document.

## Syntax

```psj
UpdateTemplate(string templateName, string comment)
```

## Inputs

### `1. string `

Template name.

### `2. string `

Comment.

## Return Code

- "1": Succeeded.
- "0": Failed.

## Sample Code

```psj
UpdateTemplate("My Template", "Comment")
```
