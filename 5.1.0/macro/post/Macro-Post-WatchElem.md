## Description

Input selected elements to Watch Data window.

## Syntax

```psj
WatchNode(cursor[] Elements)
```

## Inputs

### `1. cursor[]  `

Element list.

## Return Code

Nothing.

## Sample Code

```psj
WatchElem([193342, 193356])
```
