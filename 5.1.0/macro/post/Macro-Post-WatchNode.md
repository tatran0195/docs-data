## Description

Input selected nodes to Watch Data window.

## Syntax

```psj
WatchNode(cursor[] Nodes)
```

## Inputs

### `1. cursor[]  `

Node list.

## Return Code

Nothing.

## Sample Code

```psj
WatchNode([10:7, 10:18])
```
