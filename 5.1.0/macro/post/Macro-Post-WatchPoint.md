## Description

Input picked positions to Watch Data window.

## Syntax

```psj
WatchPoint(vector[] positions)
```

## Inputs

### `1. vector[]::double  `

x potision.

### `2. vector[]::double  `

y potision.

### `3. vector[]::double  `

z potision.

## Return Code

Nothing.

## Sample Code

```psj
WatchPoint([[0.01, 0.0078796576, 0.00270456168], [0.0052604386, 0.0099999997, 0.0033876867]])
```
