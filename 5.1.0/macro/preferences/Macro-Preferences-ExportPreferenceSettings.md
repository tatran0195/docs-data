## Description

Export settings of Preferences (.json file).

## Syntax

```psj
ExportPreferenceSettings(string filePath)
```

## Inputs

### `1. string`

Export file path.

## Return Code

No return code.

## Sample Code

```psj
ExportPreferenceSettings("C:/temp/DCADPreference.json")
```
