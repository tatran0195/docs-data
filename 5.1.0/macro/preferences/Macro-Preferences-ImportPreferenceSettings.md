## Description

Import settings of Preferences (.json file).

## Syntax

```psj
ImportPreferenceSettings(string filePath)
```

## Inputs

### `1. string`

Import file path.

## Return Code

No return code.

## Sample Code

```psj
ImportPreferenceSettings("C:/temp/DCADPreference.json")
```
