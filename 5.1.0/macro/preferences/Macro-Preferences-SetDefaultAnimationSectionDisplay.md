## Description

Select animation section display method.

## Syntax

```psj
SetDefaultAnimationSectionDisplay(int SectionDisplayType)
```

## Inputs

### `1. int`

Set animation section display method.
- 0: Lagrange Display
- 1: Euler Display

## Return Code

No return code.

## Sample Code

```psj
SetDefaultAnimationSectionDisplay(1)
```
