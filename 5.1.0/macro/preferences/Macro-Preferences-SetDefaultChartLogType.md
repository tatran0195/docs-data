## Description

Set default chart log type.

## Syntax

```psj
SetDefaultChartLogType(int LogType)
```

## Inputs

### `1. int`

Set default chart log type.
- 0: General
- 1: Log
- 2: 10log(10)
- 3: dBSPL

## Return Code

No return code.

## Sample Code

```psj
SetDefaultDataValidation(0)
```
