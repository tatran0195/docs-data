## Description

Turn ON/OFF Data Validation.

## Syntax

```psj
SetDefaultDataValidation(int Flag)
```

## Inputs

### `1. int`

Set Data Validation
- 0: OFF
- 1: ON

## Return Code

No return code.

## Sample Code

```psj
SetDefaultDataValidation(1)
```
