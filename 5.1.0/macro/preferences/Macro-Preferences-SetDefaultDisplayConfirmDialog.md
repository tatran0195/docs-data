## Description

Turn ON/OFF confirmation dialog display.

## Syntax

```psj
SetDefaultDisplayConfirmDialog(int Flag)
```

## Inputs

### `1. int`

Enables to display confiration dialog.
- 0: OFF
- 1: ON

## Return Code

No return code.

## Sample Code

```psj
SetDefaultDisplayConfirmDialog(1)
```
