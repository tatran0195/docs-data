## Description

Enalble/disable key bind

## Syntax

```psj
SetDefaultEnableKeyBind(bool flag)
```

## Inputs

### `1. bool`

- "1": Enalbe key bind
- "0": Disable key bind

## Return Code

No return code.

## Sample Code

```psj
SetDefaultEnableKeyBind(1)
```
