## Description

Turn ON/OFF the position of selection list of command dialog set to the latest position.

## Syntax

```psj
SetDefaultKeepLatestSelectionDialogPosition(int Flag)
```

## Inputs

### `1. int`

Enables to keep the position of selection list of command dialog set to the latest position.
- 0: OFF
- 1: ON

## Return Code

No return code.

## Sample Code

```psj
SetDefaultKeepLatestSelectionDialogPosition(1)
```
