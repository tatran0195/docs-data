## Description

Set common labels' display.

## Syntax

```psj
SetDisplayCommonLabels(int AxisText, int ContourLegend)
```

## Inputs

### `1. int`

Display/Hide text at Axis.
- 0: OFF
- 1: ON

### `2. int`

Display/Hide text at Contour Legend.
- 0: OFF
- 1: ON

## Return Code

No return code.

## Sample Code

```psj
SetDisplayCommonLabels(1, 1)
```
