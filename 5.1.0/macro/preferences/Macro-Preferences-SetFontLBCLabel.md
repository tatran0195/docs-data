## Description

Set font of LBC label.

## Syntax

```psj
SetFontLBCLabel(int FontSize, int Color, bool BoldType)
```

## Inputs

### `1. int`

Font size.

### `2. int`

Font color.

### `3. int`

Set bold type.
- 0: OFF
- 1: ON

## Return Code

No return code.

## Sample Code

```psj
SetFontLBCLabel(12, 255, 1)
```
