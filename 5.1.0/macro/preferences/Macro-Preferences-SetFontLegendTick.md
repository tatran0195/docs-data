## Description

Set font of legend ticks.

## Syntax

```psj
SetFontLegendTick(int FontSize, int Color, bool BoldType)
```

## Inputs

### `1. int`

Font size.

### `2. int`

Font color.

### `3. int`

Set bold type.
- 0: OFF
- 1: ON

## Return Code

No return code.

## Sample Code

```psj
SetFontLegendTick(9, 5197615, 1)
```
