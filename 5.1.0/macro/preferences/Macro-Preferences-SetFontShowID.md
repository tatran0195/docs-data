## Description

Set font of Show ID.

## Syntax

```psj
SetFontShowID(int FontSize, int Color, bool BoldType)
```

## Inputs

### `1. int`

Font size.

### `2. int`

Font color.

### `3. int`

Set bold type.
- 0: OFF
- 1: ON

## Return Code

No return code.

## Sample Code

```psj
SetFontShowID(8, 0, 0)
```
