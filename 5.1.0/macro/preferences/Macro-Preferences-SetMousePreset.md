## Description

Set current mouse preset.

## Syntax

```psj
SetMousePreset(string Preset)
```

## Inputs

### `1. String`

Name of mouse preset.

## Return Code

No return code.

## Sample Code

```psj
SetMousePreset("TSV-Pre")
```
