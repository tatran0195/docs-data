## Description

Set show/hide base unit.

## Syntax

```psj
ShowHideBaseUnit(bool bFlag)
```

## Inputs

### `1. bool`

Flag of show/hide base unit.
  - 0 : hide base unit.
  - 1 : show base unit.

## Return Code

No return code.

## Sample Code

```psj
ShowHideBaseUnit(1)
```
