## Description

Set Shell Parameter

## Syntax

```psj
ElemRelatedInfo_Shell(list[] erishell_data)
```

## Inputs

### `1. list[]`

list of erishell data

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
ElemRelatedInfo_Shell([[117, 2, 1.0472, 2147483647, 1.79769e+308], [117, 2, 1.0472, 2147483647, 1.79769e+308]])
```
