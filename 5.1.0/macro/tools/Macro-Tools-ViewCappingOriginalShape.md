## Description

Show / Hide capping face of original shape.

## Syntax

```psj
ViewCappingOriginalShape(int Flag)
```

## Inputs

### `1. int`

- "1": Show capping face of original shape.
- "0": Hide capping face of original shape.

## Return Code

Nothing.

## Sample Code

```psj
ViewCappingOriginalShape(1)
```
