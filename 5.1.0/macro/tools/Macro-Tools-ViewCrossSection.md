## Description

Show cross section cut by indicated direction.

## Syntax

```psj
ViewCrossSection(int Direction)
```

## Inputs

### `1. int`
- "0": Show cross section YZ
- "1": Show cross section XZ
- "2": Show cross section XY

## Return Code

Nothing.

## Sample Code

```psj
ViewCrossSection(1)
```
