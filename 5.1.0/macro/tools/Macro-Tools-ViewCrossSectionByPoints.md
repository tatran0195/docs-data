## Description

Show cross section cut by indicated two positions.

## Syntax

```psj
ViewCrossSectionByPoints(Vector3f point1, Vector3f point2)
```

## Inputs

### `1. Vector3f`

Position of 1st point [x,y,z]

### `2. Vector3f`

Position of 2nd point [x,y,z]

## Return Code

Nothing.

## Sample Code

```psj
ViewCrossSectionByPoints([[0.002222222222222222, 0, 0.01], [0, 0.003333333333333333, 0.01]])
```
