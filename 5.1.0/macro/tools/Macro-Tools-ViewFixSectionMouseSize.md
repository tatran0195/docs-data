## Description

Fix / free size of 3d mouse.

## Syntax

```psj
ViewFixSectionMouseSize(int Flag)
```

## Inputs

### `1. int`

- "1": Fix size of 3d mouse.
- "0": Free size of 3d mouse.

## Return Code

Nothing.

## Sample Code

```psj
ViewFixSectionMouseSize(1)
```
