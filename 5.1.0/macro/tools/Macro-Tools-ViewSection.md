## Description

Enable/Disable section display.

## Syntax

```psj
ViewSection(int Flag)
```

## Inputs

### `1. int`

- "1": Enable section display.
- "0": Disable section display.

## Return Code

Nothing.

## Sample Code

```psj
ViewSection(1)
```
