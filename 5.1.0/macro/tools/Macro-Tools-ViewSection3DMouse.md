## Description

Show/hide 3d mouse view.

## Syntax

```psj
ViewSection3DMouse(int Flag)
```

## Inputs

### `1. int`

- "1": Show 3d mouse.
- "0": Hide 3d mouse.

## Return Code

Nothing.

## Sample Code

```psj
ViewSection3DMouse(1)
```
