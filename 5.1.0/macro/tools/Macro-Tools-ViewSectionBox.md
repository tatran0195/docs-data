## Description

Turn on/off box view.

## Syntax

```psj
ViewSectionBox(int Flag)
```

## Inputs

### `1. int`

- "1": Turn on box view.
- "0": Turn off box view.

## Return Code

Nothing.

## Sample Code

```psj
ViewSectionBox(1)
```
