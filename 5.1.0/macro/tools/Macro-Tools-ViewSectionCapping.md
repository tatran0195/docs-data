## Description

Show/hide capping face of cross section.

## Syntax

```psj
ViewSectionCapping(int Flag)
```

## Inputs

### `1. int`

- "1": Show capping face.
- "0": Hide capping face.

## Return Code

Nothing.

## Sample Code

```psj
ViewSectionCapping(1)
```
