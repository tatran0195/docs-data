## Description

Show / Hide cutting edge.

## Syntax

```psj
ViewSectionCuttingEdge(int Flag)
```

## Inputs

### `1. int`

- "1": Show cutting edge.
- "0": Hide cutting edge.

## Return Code

Nothing.

## Sample Code

```psj
ViewSectionCuttingEdge(1)
```
