## Description

Show / Hide section elements.

## Syntax

```psj
ViewSectionElements(int Flag)
```

## Inputs

### `1. int`

- "1": Show section elements.
- "0": Hide section elements.

## Return Code

Nothing.

## Sample Code

```psj
ViewSectionElements(1)
```
