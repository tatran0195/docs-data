## Description

Show / Hide mesh line at section.

## Syntax

```psj
ViewSectionMeshLine(int Flag)
```

## Inputs

### `1. int`

- "1": Show mesh line at section.
- "0": Hide mesh line at section.

## Return Code

Nothing.

## Sample Code

```psj
ViewSectionMeshLine(1)
```
