## Description

Turn on/off transparency of display of hidden side.

## Syntax

```psj
ViewSectionTransparency(int Flag)
```

## Inputs

### `1. int`

- "1": Turn on transparency of display of hidden side.
- "0": Turn off transparency display of hidden side.

## Return Code

Nothing.

## Sample Code

```psj
ViewSectionTransparency(1)
```
