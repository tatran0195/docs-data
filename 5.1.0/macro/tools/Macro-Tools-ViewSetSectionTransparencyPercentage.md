## Description

Set transparency of hidden side if section transparency turns ON.

## Syntax

```psj
ViewSetSectionTransparencyPercentage(int percentage)
```

## Inputs

### `1. Int`

Percentage of transparency.

## Return Code

Nothing.

## Sample Code

```psj
ViewSetSectionTransparencyPercentage(50)
```
