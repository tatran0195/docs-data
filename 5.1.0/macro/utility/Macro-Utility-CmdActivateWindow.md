## Description

Activate a specific Pre or Post window

## Syntax

```psj
CmdActivateWindow(String Name)
```

## Inputs

### `1. String`

Window name

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
CmdActivateWindow("Jupiter1")
```
