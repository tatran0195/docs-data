## Description

Find Maximum Area Face in a Part

## Syntax

```psj
FindMaxAreaFaceInPart(cursor crTargetPart)
```

## Inputs

### `3. Cursor`

Target part

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
FindMaxAreaFaceInPart(3:4)
```
