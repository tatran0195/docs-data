## Description

Find Minimum Area Face in a Part

## Syntax

```psj
FindMinAreaFaceInPart(Cursor crTargetPart)
```

## Inputs

### `3. Cursor`

Target part

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
FindMinAreaFaceInPart(3:4)
```
