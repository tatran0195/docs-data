## Description

Get Part Name From Keys

## Syntax

```psj
GetPartNameFromKey(ID[])
```

## Inputs

### `1. ID[]`

Part ID(s)

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
GetPartNameFromKey([1,2,5])
```
