## Description

Print strings on console. This function is mainly used in background mode to display arbitrary strings on console.

## Syntax

```psj
PrintJPT(string String)
```

## Inputs

### `1. String`

String to display

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
PrintJPT("test")
```
