## Description

repeat the operation

## Syntax

```psj
Redo(int iRedoTimes)
```

## Inputs

### `1. Int`

Redo times

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
Redo(1)
```
