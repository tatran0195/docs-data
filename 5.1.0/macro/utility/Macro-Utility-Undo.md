## Description

Cancel the operation

## Syntax

```psj
Undo(int iUndoTimes)
```

## Inputs

### `1. Int`

Undo times

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
Undo(1)
```
