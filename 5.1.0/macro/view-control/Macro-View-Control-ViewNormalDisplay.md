## Description

Back to the Normal Display.

## Syntax

```psj
ViewNormalDisplay()
```

## Inputs

No Arguments

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
ViewNormalDisplay()
```
