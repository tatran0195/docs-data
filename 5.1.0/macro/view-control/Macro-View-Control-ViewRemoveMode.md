## Description

Turn Remove Mode ON/OFF

## Syntax

```psj
ViewRemoveMode(bool Flag)
```

## Inputs

### `1. bool`

Turn ON/OFF Remove Mode.

- 0: OFF
- 1: ON

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
ViewRemoveMode(1)
```
