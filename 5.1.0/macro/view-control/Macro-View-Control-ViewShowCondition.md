## Description

Turn ON/OFF Show Condition

## Syntax

```psj
ViewShowCondition(bool Flag)
```

## Inputs

1\. bool
Turn on/off display switch 0=On 1=OFF

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
ViewShowCondition(0)
```
