## Description

Turn ON/OFF Group Mode

## Syntax

```psj
ViewShowGroupMode(bool Flag)
```

## Inputs

### `1. Bool`

Turn on/off group mode switch 0=On 1=OFF

## Return Code

- "1": Turn on group mode
- "0": Turn off group mode

## Sample Code

```psj
ViewShowGroupMode(0)
```
