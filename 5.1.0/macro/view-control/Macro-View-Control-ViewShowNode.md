## Description

Turn ON/OFF Show Node

## Syntax

```psj
ViewShowNode()
```

## Inputs

### `1. Bool`

Turn on/off display switch 0=On 1=OFF

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
ViewShowNode(0)
```
