## Description

Turn ON/OFF Show Origin

## Syntax

```psj
ViewShowOrigin(bool flag)
```

## Inputs

### `1. Bool`

Turn on/off display switch 0=OFF 1=ON

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
ViewShowOrigin(0)
```
