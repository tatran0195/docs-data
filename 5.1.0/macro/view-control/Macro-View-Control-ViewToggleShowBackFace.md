## Description

Turn ON/OFF Show Back Face as the function called.

## Syntax

```psj
ViewToggleShowBackFace()
```

## Inputs

No Arguments

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
ViewToggleShowBackFace()
```
