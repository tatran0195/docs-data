## Description

Turn ON/OFF Show Bar Shading as the function called.

## Syntax

```psj
ViewToggleShowBarShading()
```

## Inputs

No Arguments

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
ViewToggleShowBarShading()
```
