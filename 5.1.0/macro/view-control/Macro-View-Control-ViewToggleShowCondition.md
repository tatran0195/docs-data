## Description

Turn ON/OFF Show Condition as the function called.

## Syntax

```psj
ViewToggleShowCondition()
```

## Inputs

No Arguments

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
ViewToggleShowCondition()
```
