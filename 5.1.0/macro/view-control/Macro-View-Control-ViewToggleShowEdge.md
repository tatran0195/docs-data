## Description

Turn ON/OFF Show Edge as the function called.

## Syntax

```psj
ViewToggleShowEdge()
```

## Inputs

No Arguments

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
ViewToggleShowEdge()
```
