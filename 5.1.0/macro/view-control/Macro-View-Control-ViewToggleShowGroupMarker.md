## Description

Turn ON/OFF Show Group Marker as the function called.

## Syntax

```psj
ViewToggleShowGroupMarker()
```

## Inputs

No Arguments

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
ViewToggleShowGroupMarker()
```
