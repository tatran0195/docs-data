## Description

Turn ON/OFF Show ID as the function called.

## Syntax

```psj
ViewToggleShowID()
```

## Inputs

No Arguments

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
ViewToggleShowID()
```
