## Description

Turn ON/OFF Show Local Setting as the function called.

## Syntax

```psj
ViewToggleShowLocalSetting()
```

## Inputs

No Arguments

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
ViewToggleShowLocalSetting()
```
