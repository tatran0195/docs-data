## Description

Turn ON/OFF Show Mesh as the function called.

## Syntax

```psj
ViewToggleShowMesh()
```

## Inputs

No Arguments

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
ViewToggleShowMesh()
```
