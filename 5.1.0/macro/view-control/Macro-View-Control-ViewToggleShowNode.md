## Description

Turn ON/OFF Show Node as the function called.

## Syntax

```psj
ViewToggleShowNode()
```

## Inputs

No Arguments

## Return Code

- "1": The function can be executed
- "0": The function cannot be executed

## Sample Code

```psj
ViewToggleShowNode()
```
