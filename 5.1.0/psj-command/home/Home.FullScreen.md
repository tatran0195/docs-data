## Description

Changes Main Window to full screen display.

## Syntax

```psj
Home.FullScreen()
```

Ribbon: <menuselection>Home &#187; FullScreen</menuselection>

## Inputs

None

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
Home.FullScreen()
```
