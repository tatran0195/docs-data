## Description

## Syntax

```psj
MeshCleanup.Manual3D.Equivalence(crlNodes=[], iMergeTowards=0)
```

Ribbon: <menuselection>MeshCleanup &#187; Manual3D &#187; Equivalence</menuselection>

## Inputs

### `crlNodes`

- A _List of Cursor_ specifying the node.
- The default value is [].

### `iMergeTowards`

- An _Integer_ specifying the merge towards.
- The default value is 0.

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
MeshCleanup.Manual3D.Equivalence(crlNodes=[], iMergeTowards=0)
```
