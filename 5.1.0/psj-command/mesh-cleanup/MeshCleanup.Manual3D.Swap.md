## Description

Cleanup element edge by swap

## Syntax

```psj
MeshCleanup.Manual3D.Swap(crplElemEdge)
```

Ribbon: <menuselection>MeshCleanup &#187; Manual3D &#187; Swap</menuselection>

## Inputs

### `crplElemEdge`

- A _Cursor Pair List_ specifying the element edge.
- This is a required input.

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
MeshCleanup.Manual3D.Swap(crplElemEdge)
```
