## Description

Element Conversion

## Syntax

```psj
MeshEdit.ElementConvert(crlParts=[], iType=1)
```

Ribbon: <menuselection>MeshEdit &#187; ElementConvert</menuselection>

## Inputs

### `crlParts`

- A _List of Cursor_ specifying the part.
- The default value is [].

### `iType`

- An _Integer_ specifying the type.
- The default value is 1.

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
MeshEdit.ElementConvert(crlParts=[], iType=1)
```
