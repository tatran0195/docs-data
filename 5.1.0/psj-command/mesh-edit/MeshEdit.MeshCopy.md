## Description

Mesh Copy Pattern

## Syntax

```psj
MeshEdit.MeshCopy(crlFaces=[], crlNodes=[])
```

Ribbon: <menuselection>MeshEdit &#187; MeshCopy</menuselection>

## Inputs

### `crlFaces`

- A _List of Cursor_ specifying the face.
- The default value is [].

### `crlNodes`

- A _List of Cursor_ specifying the node.
- The default value is [].

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
MeshEdit.MeshCopy(crlFaces=[], crlNodes=[])
```
