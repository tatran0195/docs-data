## Description

Mirror copy of surface mesh

## Syntax

```psj
MeshEdit.MirrorCopy(crlFaces=[])
```

Ribbon: <menuselection>MeshEdit &#187; MirrorCopy</menuselection>

## Inputs

### `crlFaces`

- A _List of Cursor_ specifying the face.
- The default value is [].

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
MeshEdit.MirrorCopy(crlFaces=[])
```
