## Description

## Syntax

```psj
MeshEdit.MoveNode.CADFollows(crlNodes=[], dMovedPosX=0.0, dMovedPosY=0.0, dMovedPosZ=0.0)
```

Ribbon: <menuselection>MeshEdit &#187; MoveNode &#187; CADFollows</menuselection>

## Inputs

### `crlNodes`

- A _List of Cursor_ specifying the node.
- The default value is [].

### `dMovedPosX`

- A _Double_ specifying the moved position x.
- The default value is 0.0.

### `dMovedPosY`

- A _Double_ specifying the moved position y.
- The default value is 0.0.

### `dMovedPosZ`

- A _Double_ specifying the moved position z.
- The default value is 0.0.

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
MeshEdit.MoveNode.CADFollows(crlNodes=[], dMovedPosX=0.0, dMovedPosY=0.0, dMovedPosZ=0.0)
```
