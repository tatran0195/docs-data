## Description

Move node by equalize

## Syntax

```psj
MeshEdit.MoveNode.Equalize(crlEdges=[])
```

Ribbon: <menuselection>MeshEdit &#187; MoveNode &#187; Equalize</menuselection>

## Inputs

### `crlEdges`

- A _List of Cursor_ specifying the edge.
- The default value is [].

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
MeshEdit.MoveNode.Equalize(crlEdges=[])
```
