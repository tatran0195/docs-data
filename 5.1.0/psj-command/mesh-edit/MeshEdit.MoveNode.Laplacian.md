## Description

## Syntax

```psj
MeshEdit.MoveNode.Laplacian(crlTargets=[], iType=0, bWithCADFollow=False)
```

Ribbon: <menuselection>MeshEdit &#187; MoveNode &#187; Laplacian</menuselection>

## Inputs

### `crlTargets`

- A _List of Cursor_ specifying the target.
- The default value is [].

### `iType`

- An _Integer_ specifying the type.
- The default value is 0.

### `bWithCADFollow`

- A _Boolean_ specifying the with CAD follow.
- The default value is False.

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
MeshEdit.MoveNode.Laplacian(crlTargets=[], iType=0, bWithCADFollow=False)
```
