## Description

Move Nodes from Node to 3 nodes created Plane

## Syntax

```psj
MeshEdit.MoveNode.ProjectToPlane_3Nodes(ilNodeList=[])
```

Ribbon: <menuselection>MeshEdit &#187; MoveNode &#187; ProjectToPlane_3Nodes</menuselection>

## Inputs

### `ilNodeList`

- A _List of Integer_ specifying the node list.
- The default value is [].

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
MeshEdit.MoveNode.ProjectToPlane_3Nodes(ilNodeList=[])
```
