## Description

Exent face to face

## Syntax

```psj
MidPlaneEdit.AddItems.Edge.FaceTwoFace(crRefFace=None, crExtFace=None, iExtendType=0)
```

Ribbon: <menuselection>MidPlaneEdit &#187; AddItems &#187; Edge &#187; FaceTwoFace</menuselection>

## Inputs

### `crRefFace`

- A _Cursor_ specifying the reference face.
- The default value is None.

### `crExtFace`

- A _Cursor_ specifying the extend face.
- The default value is None.

### `iExtendType`

- An _Integer_ specifying the extend type.
- The default value is 0.

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
MidPlaneEdit.AddItems.Edge.FaceTwoFace(crRefFace=None, crExtFace=None, iExtendType=0)
```
