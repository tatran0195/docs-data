## Description

Edit mid-plane with edge nodes

## Syntax

```psj
MidPlaneEdit.Edge.Nodes(crlNodes=[])
```

Ribbon: <menuselection>MidPlaneEdit &#187; Edge &#187; Nodes</menuselection>

## Inputs

### `crlNodes`

- A _List of Cursor_ specifying the node.
- The default value is [].

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
MidPlaneEdit.Edge.Nodes(crlNodes=[])
```
