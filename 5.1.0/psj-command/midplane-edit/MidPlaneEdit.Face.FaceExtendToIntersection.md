## Description

Face Extend To Intersection

## Syntax

```psj
MidPlaneEdit.Face.FaceExtendToIntersection(crEdge0, crEdge1)
```

Ribbon: <menuselection>MidPlaneEdit &#187; Face &#187; FaceExtendToIntersection</menuselection>

## Inputs

### `crEdge0`

- A _Cursor_ specifying the edge0.
- This is a required input.

### `crEdge1`

- A _Cursor_ specifying the edge1.
- This is a required input.

## Return Code

A String of 1 if success, or 0 if fail.
