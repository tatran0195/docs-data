## Description

Face Cross

## Syntax

```psj
MidPlane.FaceCross(crlBodies=[], crlFaces=[])
```

Ribbon: <menuselection>MidPlane &#187; FaceCross</menuselection>

## Inputs

### `crlBodies`

- A _List of Cursor_ specifying the bodies.
- The default value is [].

### `crlFaces`

- A _List of Cursor_ specifying the faces.
- The default value is [].

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
MidPlane.FaceCross(crlBodies=[], crlFaces=[])
```
