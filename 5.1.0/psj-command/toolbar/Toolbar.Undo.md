## Description

Undo

## Syntax

```psj
Toolbar.Undo(iCntUndo=1)
```

Ribbon: <menuselection>Toolbar &#187; Undo</menuselection>

## Inputs

### `iCntUndo`

- An _Integer_ specifying the cnt undo.
- The default value is 1.

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
Toolbar.Undo(iCntUndo=1)
```
