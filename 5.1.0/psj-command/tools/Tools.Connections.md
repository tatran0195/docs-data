## Description

Renumber connection

## Syntax

```psj
Tools.Connections(listConnectRenumberTool)
```

Ribbon: <menuselection>Tools &#187; Connections</menuselection>

## Inputs

### `listConnectRenumberTool`

- A _CONNECT_RENUMBER_TOOL List_ specifying the connect renumber tool.
- This is a required input.

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
Tools.Connections(listConnectRenumberTool)
```
