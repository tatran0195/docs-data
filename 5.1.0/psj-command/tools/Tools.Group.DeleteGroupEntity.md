## Description

Delete Entity in Group

## Syntax

```psj
Tools.Group.DeleteGroupEntity(crlGroups)
```

Ribbon: <menuselection>Tools &#187; Group &#187; DeleteGroupEntity</menuselection>

## Inputs

### `crlGroups`

- A _List of Cursor_ specifying the del group.
- This is a required input.

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
Tools.Group.DeleteGroupEntity(crlGroups)
```
