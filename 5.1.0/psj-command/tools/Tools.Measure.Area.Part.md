## Description

Measure Distance By FaceNode

## Syntax

```psj
Tools.Measure.Area.Part(crlParts=[], iPrecision=6)
```

Ribbon: <menuselection>Tools &#187; Measure &#187; Area &#187; Part</menuselection>

## Inputs

### `crlParts`

- A _List of Cursor_ specifying the part.
- The default value is [].

### `iPrecision`

- An _Integer_ specifying the precision.
- The default value is 6.

## Return Code

A _Double_ specifying the area value of a part or a total area value of parts.

## Sample Code

```psj
Tools.Measure.Area.Part(crlParts=[], iPrecision=6)
```
