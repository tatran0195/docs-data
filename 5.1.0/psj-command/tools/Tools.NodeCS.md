## Description

Create Node CS

## Syntax

```psj
Tools.NodeCS(crlInst=[], crCoordSystem=None)
```

Ribbon: <menuselection>Tools &#187; NodeCS</menuselection>

## Inputs

### `crlInst`

- A _List of Cursor_ specifying the inst.
- The default value is [].

### `crCoordSystem`

- A _Cursor_ specifying the coordinate system.
- The default value is None.

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
Tools.NodeCS(crlInst=[], crCoordSystem=None)
```
