## Description

Enable / Disable section display.

## Syntax

```psj
Tools.Section(bSection)
```

Ribbon: <menuselection>Tools &#187; Section</menuselection>

## Inputs

### `bSection`

- A _Boolean_ specifying the section.
- This is a required input.

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
Tools.Section(bSection=True)
```
