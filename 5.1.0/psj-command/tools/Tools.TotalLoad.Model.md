## Description

## Syntax

```psj
Tools.TotalLoad.Model(crlTargets=[], crCoordinate=None, strOutput="Total", iPrecision=6)
```

Ribbon: <menuselection>Tools &#187; TotalLoad &#187; Model</menuselection>

## Inputs

### `crlTargets`

- A _List of Cursor_ specifying the target.
- The default value is [].

### `crCoordinate`

- A _Cursor_ specifying the coordinate.
- The default value is None.

### `strOutput`

- A _String_ specifying the output.
- The default value is "Total".

### `iPrecision`

- An _Integer_ specifying the precision.
- The default value is 6.

## Return Code

A String of 1 if success, or 0 if fail.

## Sample Code

```psj
Tools.TotalLoad.Model(crlTargets=[], crCoordinate=None, strOutput="Total", iPrecision=6)
```
