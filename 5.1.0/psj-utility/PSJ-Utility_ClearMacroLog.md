## Description

Clear all the text existing in the Macro window.

## Syntax

```psj
JPT.ClearMacroLog()
```

## Inputs

This utility function does not require any input value.

## Return Code

This utility function does not have output value.

## Sample Code

```psj {2}
# Clear all the text existing on the Macro window
JPT.ClearMacroLog()
```
