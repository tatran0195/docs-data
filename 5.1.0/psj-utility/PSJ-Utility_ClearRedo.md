## Description

Clear all the saved redo steps.

## Syntax

```psj
JPT.ClearRedo()
```

## Inputs

This utility function does not require any input value.

## Return Code

This utility function does not have output value.

## Sample Code

```psj {2}
# Clear all the saved redo steps
JPT.ClearRedo()
```
