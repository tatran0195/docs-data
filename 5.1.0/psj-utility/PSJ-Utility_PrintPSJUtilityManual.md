## Description

Print all the information of all PSJ utilities to the Python API window.

## Syntax

```psj
JPT.PrintPSJUtilityManual()
```

## Inputs

This utility function does not require any input value.

## Return Code

This utility function does not have output value.

## Sample Code

```psj {2}
# Print all the information of all PSJ utilities to the Python API Window
JPT.PrintPSJUtilityManual()
```
