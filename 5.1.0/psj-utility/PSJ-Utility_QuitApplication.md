## Description

Quit Jupiter immediately.

## Syntax

```psj
JPT.QuitApplication()
```

## Inputs

This utility function does not require any input value.

## Return Code

This utility function does not have output value.

## Sample Code

```psj {2}
# Quit Jupiter immediately
JPT.QuitApplication()
```
