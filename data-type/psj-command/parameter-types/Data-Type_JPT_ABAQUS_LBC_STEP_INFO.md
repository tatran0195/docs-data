## Description

A data type uses to control parameters of Abaqus Lbc Step Info

## Attributes

### `crStep`

- A _Cursor_ specifying step.
- The default value is None.

### `crLbc`

- A _Cursor_ specifying lbc.
- The default value is None.

### `iCurflag`

- An _Integer_ specifying curflag.
- The default value is 0.
