## Description

A data type uses to control parameters of Abaqus Output Request

## Attributes

### `strName`

- A _String_ specifying name.
- The default value is "".

### `strDescription`

- A _String_ specifying description.
- The default value is "".

### `iId`

- An _Integer_ specifying ID.
- The default value is 0.

### `iOutputFiletype`

- An _Integer_ specifying output filetype.
- The default value is 0.

### `iDomainType`

- An _Integer_ specifying domain type.
- The default value is 0.

### `iDomainId`

- An _Integer_ specifying domain ID.
- The default value is 0.

### `iFrequencyType`

- An _Integer_ specifying frequency type.
- The default value is 0.

### `dFrequency`

- A _Double_ specifying frequency.
- The default value is 0.0.

### `ParamTimePoint`

- A _PARAM_TIME_POINT_ specifying time point.
- The default value is TIME_POINTS().

### `iTiming`

- An _Integer_ specifying timing.
- The default value is 0.
