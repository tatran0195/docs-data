## Description

A data type uses to control parameters of ADVC Auto Increment

## Attributes

### `iNewtonResidueIncrMax`

- An _Integer_ specifying newton residue incr maximum.
- The default value is DFLT_INT.

### `iBeginResidueIncrCheck`

- An _Integer_ specifying begin residue incr check.
- The default value is DFLT_INT.

### `iBeginLogarithmicRateCheck`

- An _Integer_ specifying begin logarithmic rate check.
- The default value is DFLT_INT.

### `dCutBackFactorForDivergence`

- A _Double_ specifying cut back factor for divergence.
- The default value is DFLT_DBL.

### `dCutBackFactorForTooSlowConvergence`

- A _Double_ specifying cut back factor for too slow convergence.
- The default value is DFLT_DBL.

### `dCutBackFactorForExcessiveDistortion`

- A _Double_ specifying cut back factor for excessive distortion.
- The default value is DFLT_DBL.

### `iIncrEnlargeNewtonMax`

- An _Integer_ specifying incr enlarge newton maximum.
- The default value is DFLT_INT.

### `iIncrEnlargeSuppress`

- An _Integer_ specifying incr enlarge suppress.
- The default value is DFLT_INT.

### `dIncreaseFactorForStatic`

- A _Double_ specifying increase factor for static.
- The default value is DFLT_DBL.

### `dIncreaseFactorForDynamic`

- A _Double_ specifying increase factor for dynamic.
- The default value is DFLT_DBL.

### `dIncreaseFactorForCreep`

- A _Double_ specifying increase factor for creep.
- The default value is DFLT_DBL.

### `dIncreaseFactorForRdnlk`

- A _Double_ specifying increase factor for rdnlk.
- The default value is DFLT_DBL.

### `dTemperatureIncrMax`

- A _Double_ specifying temperature incr maximum.
- The default value is DFLT_DBL.

### `iUseTemperatureIncrMax`

- An _Integer_ specifying use temperature incr maximum.
- The default value is 0.

### `dHalfStepTol`

- A _Double_ specifying half step tolerance.
- The default value is DFLT_DBL.

### `dStraTol`

- A _Double_ specifying stra tolerance.
- The default value is DFLT_DBL.

### `dCreepStraTol`

- A _Double_ specifying creep stra tolerance.
- The default value is DFLT_DBL.
