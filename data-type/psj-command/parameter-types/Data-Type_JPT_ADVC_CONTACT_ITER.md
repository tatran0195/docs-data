## Description

A data type uses to control parameters of ADVC Contact Iter

## Attributes

### `iSubdivideMode`

- An _Integer_ specifying subdivide mode.
- The default value is -1.

### `iContactloopMax`

- An _Integer_ specifying contactloop maximum.
- The default value is DFLT_INT.

### `iInternalContactloopMax`

- An _Integer_ specifying internal contactloop maximum.
- The default value is DFLT_INT.

### `dSeparationTol`

- A _Double_ specifying separation tolerance.
- The default value is DFLT_DBL.

### `dRelativeSeparationTol`

- A _Double_ specifying relative separation tolerance.
- The default value is DFLT_DBL.

### `dPenetrationTol`

- A _Double_ specifying penetration tolerance.
- The default value is DFLT_DBL.

### `dRelativePenetrationTol`

- A _Double_ specifying relative penetration tolerance.
- The default value is DFLT_DBL.

### `iMaxchp`

- An _Integer_ specifying maxchp.
- The default value is DFLT_INT.

### `iFrictionOnset`

- An _Integer_ specifying friction onset.
- The default value is 0.

### `dStickSlipTol`

- A _Double_ specifying stick slip tolerance.
- The default value is DFLT_DBL.

### `dFrictionTol`

- A _Double_ specifying friction tolerance.
- The default value is DFLT_DBL.
