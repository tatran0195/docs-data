## Description

A data type uses to control parameters of ADVC Convergence

## Attributes

### `dCgTol`

- A _Double_ specifying cg tolerance.
- The default value is DFLT_DBL.

### `dCgNrTol`

- A _Double_ specifying cg nr tolerance.
- The default value is DFLT_DBL.

### `dCgDispTol`

- A _Double_ specifying cg displacement tolerance.
- The default value is DFLT_DBL.

### `dCgNrDispTol`

- A _Double_ specifying cg nr displacement tolerance.
- The default value is DFLT_DBL.

### `dCgDispLimitTol`

- A _Double_ specifying cg displacement limit tolerance.
- The default value is DFLT_DBL.

### `dCgTotalDispLimitTol`

- A _Double_ specifying cg total displacement limit tolerance.
- The default value is DFLT_DBL.

### `dNewtonTol`

- A _Double_ specifying newton tolerance.
- The default value is DFLT_DBL.

### `dNewtonDispTol`

- A _Double_ specifying newton displacement tolerance.
- The default value is DFLT_DBL.

### `dNewtonDispLimitTol`

- A _Double_ specifying newton displacement limit tolerance.
- The default value is DFLT_DBL.

### `dNewtonTotalDispLimitTol`

- A _Double_ specifying newton total displacement limit tolerance.
- The default value is DFLT_DBL.

### `iCgloopMax`

- An _Integer_ specifying cgloop maximum.
- The default value is DFLT_INT.

### `iNewtonMax`

- An _Integer_ specifying newton maximum.
- The default value is DFLT_INT.

### `dHtNlLoopTol`

- A _Double_ specifying ht nl loop tolerance.
- The default value is DFLT_DBL.
