## Description

A data type uses to control parameters of ADVC Heat Time Step

## Attributes

### `iEndType`

- An _Integer_ specifying end type.
- The default value is 1.

### `dMaxTime`

- A _Double_ specifying maximum time.
- The default value is 1.

### `iFixedAuto`

- An _Integer_ specifying fixed auto.
- The default value is 0.

### `dMaxChange`

- A _Double_ specifying maximum change.
- The default value is DFLT_DBL.

### `dInitdt`

- A _Double_ specifying initdt.
- The default value is DFLT_DBL.

### `iDefineMaxdt`

- An _Integer_ specifying define maxdt.
- The default value is 0.

### `dMaxdt`

- A _Double_ specifying maxdt.
- The default value is DFLT_DBL.

### `iDefineMindt`

- An _Integer_ specifying define mindt.
- The default value is 0.

### `dMindt`

- A _Double_ specifying mindt.
- The default value is DFLT_DBL.

### `dFixeddt`

- A _Double_ specifying fixeddt.
- The default value is DFLT_DBL.

### `iOutputLast`

- An _Integer_ specifying output last.
- The default value is -1.

### `iOutputInterval`

- An _Integer_ specifying output interval.
- The default value is DFLT_INT.

### `iRestartLast`

- An _Integer_ specifying restart last.
- The default value is -1.

### `iRestartInterval`

- An _Integer_ specifying restart interval.
- The default value is DFLT_INT.

### `dOutputTimeInterval`

- A _Double_ specifying output time interval.
- The default value is DFLT_DBL.

### `dRestartTimeInterval`

- A _Double_ specifying restart time interval.
- The default value is DFLT_DBL.

### `iOutputInit`

- An _Integer_ specifying output init.
- The default value is -1.
