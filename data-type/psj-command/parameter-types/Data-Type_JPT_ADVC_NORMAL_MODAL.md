## Description

A data type uses to control parameters of ADVC Normal Modal

## Attributes

### `iNumberModes`

- An _Integer_ specifying number modes.
- The default value is 2.

### `iEigenNormalizationMethod`

- An _Integer_ specifying eigen normalization method.
- The default value is -1.

### `dShift`

- A _Double_ specifying shift.
- The default value is DFLT_DBL.

### `dCGCGPITolerance`

- A _Double_ specifying c g c g p i tolerance.
- The default value is DFLT_DBL.

### `dCGCGPIEigenTolerance`

- A _Double_ specifying c g c g p i eigen tolerance.
- The default value is DFLT_DBL.

### `iCGCGPILoopMax`

- An _Integer_ specifying c g c g p i loop maximum.
- The default value is DFLT_INT.

### `dCGCGPIInnerTolerance`

- A _Double_ specifying c g c g p i inner tolerance.
- The default value is DFLT_DBL.

### `iCGCGPIBlockSize`

- An _Integer_ specifying c g c g p i block size.
- The default value is DFLT_INT.
