## Description

A data type uses to control parameters of ADVC Ref Stress Result

## Attributes

### `iProc`

- An _Integer_ specifying proc.
- The default value is DFLT_INT.

### `iStep`

- An _Integer_ specifying step.
- The default value is DFLT_INT.
