## Description

A data type uses to control parameters of ADVC Struct Time Step

## Attributes

### `iFixedOrAuto`

- An _Integer_ specifying fixed or auto.
- The default value is 0.

### `iNumOfInc`

- An _Integer_ specifying number of increasement.
- The default value is 1.

### `dMaxTime`

- A _Double_ specifying maximum time.
- The default value is 1.0.

### `dMaxDt`

- A _Double_ specifying maximum dt.
- The default value is DFLT_DBL.

### `dMinDt`

- A _Double_ specifying minimum dt.
- The default value is DFLT_DBL.

### `iLoadType`

- An _Integer_ specifying load type.
- The default value is -1.

### `iOutputLast`

- An _Integer_ specifying output last.
- The default value is -1.

### `iOutputInterval`

- An _Integer_ specifying output interval.
- The default value is DFLT_INT.

### `iRestartLast`

- An _Integer_ specifying restart last.
- The default value is -1.

### `iRestartInterval`

- An _Integer_ specifying restart interval.
- The default value is DFLT_INT.

### `dOutputTimeInterval`

- A _Double_ specifying output time interval.
- The default value is DFLT_DBL.
