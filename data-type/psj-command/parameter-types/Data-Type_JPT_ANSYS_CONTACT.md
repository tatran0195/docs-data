## Description

A data type uses to control parameters of Ansys Contact

## Attributes

### `dFricCoef`

- A _Double_ specifying fric coeffience.
- The default value is DFLT_DBL.

### `dPenaStiffness`

- A _Double_ specifying pena stiffness.
- The default value is DFLT_DBL.

### `dPetrTolerance`

- A _Double_ specifying petr tolerance.
- The default value is DFLT_DBL.

### `iStiffUpdate`

- An _Integer_ specifying stiff update.
- The default value is 2.

### `iAlgorithm`

- An _Integer_ specifying algorithm.
- The default value is 2.

### `iDetec`

- An _Integer_ specifying detec.
- The default value is 2.

### `iBehavior`

- An _Integer_ specifying behavior.
- The default value is 5.
