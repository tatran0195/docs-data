## Description

A data type uses to control parameters of Basic

## Attributes

### `bBasicOutputDisplacements`

- A _Boolean_ enable/disable basic output displacements.
- The default value is False.

### `bOutputReactionLoad`

- A _Boolean_ enable/disable output reaction load.
- The default value is False.

### `bBasicOutputStrain`

- A _Boolean_ enable/disable basic output strain.
- The default value is False.

### `bBasicOutputStress`

- A _Boolean_ enable/disable basic output stress.
- The default value is False.

### `iAnalysisOption`

- An _Integer_ specifying analysis option.
- The default value is 0.

### `bCalculatePrestressEffect`

- A _Boolean_ enable/disable calculate prestress effect.
- The default value is False.

### `dUniformTemperature`

- A _Double_ specifying uniform temperature.
- The default value is 0.0.

### `dReferenceTemperature`

- A _Double_ specifying reference temperature.
- The default value is 0.0.

### `dEndLoadTime`

- A _Double_ specifying end load time.
- The default value is 0.0.

### `iTimeStep`

- An _Integer_ specifying time step.
- The default value is 0.

### `iStepChosen`

- An _Integer_ specifying step chosen.
- The default value is 0.

### `iSubStepNum`

- An _Integer_ specifying sub step number.
- The default value is 0.

### `iMaxSubStep`

- An _Integer_ specifying maximum sub step.
- The default value is 0.

### `iMinStepNum`

- An _Integer_ specifying minimum step number.
- The default value is 0.

### `dTimeStepSize`

- A _Double_ specifying time step size.
- The default value is 0.0.

### `dMinTimeStep`

- A _Double_ specifying minimum time step.
- The default value is 0.0.

### `dMaxTimeStep`

- A _Double_ specifying maximum time step.
- The default value is 0.0.

### `iWriteResultFre`

- An _Integer_ specifying write result frequency.
- The default value is 1.
