## Description

A data type uses to control parameters of bolt hole face. 

## Attributes

### `crlMasterFaces`

- A _list of Cursor_ specifying master faces.

### `crlSlaveFaces`

- A _list of Cursor_ specifying slave faces.

### `dlCenterPoint`

- A _Double Vector_ specifying center point.

### `dlMasterPoint`

- A _Double Vector_ specifying center point.

### `dlSlavePoint`

- A _Double Vector_ specifying center point.

### `iBoltType`

- An _Int_ specifying Bolt Type. 
  - 0: Type A
  - 1: Type B
  - 2: Type C
  - 3: Type D
