## Description

A data type uses to control parameters of bolt type D

## Attributes

### `iConType`
- An _Int_ specifying connection type.
  - 0: RBE2
  - 1: RBE3
- The default value is 0.

### `iConCase`
- An _Int_ specifying connection setting.
  - 0: By diameter.
  - 1: By Layer.
- The default value is 1.

### `dConDiamter`
- A _Double_ specifying Connection Diameter(Node).
- The default value is 0.02.

### `iConLayer`
- An _Int_ specifying Num of Layer of Connection Layer(Face).
- The default value is 1.
