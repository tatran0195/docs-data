## Description

A data type uses to control the note color based on user input values.

## Attributes
  
### `dUserMax`
- A _Double_ specifying the maximum user value.
- The default value is 0.0.
  
### `dUserMin`
- A _Double_ specifying the maximum user value.
- The default value is 0.0.

### `iUserMaxColor`
- An _Integer_ specifying the note color for peaks greater or equal to the specified maximum value.
- The default value is 255.

### `iUserMinColor`
- A _Integer_ specifying the note color for peaks less or equal to the specified minimum value.
- The default value is 16711680.
