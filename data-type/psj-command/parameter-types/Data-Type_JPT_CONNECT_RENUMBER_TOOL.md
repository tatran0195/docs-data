## Description

A data type uses to control parameters of Connect Renumber Tool

## Attributes

### `iItemType`

- An _Integer_ specifying item type.
- The default value is 0.

### `iKConn`

- An _Integer_ specifying k connection.
- The default value is 0.

### `iSolver`

- An _Integer_ specifying solver.
- The default value is 0.
