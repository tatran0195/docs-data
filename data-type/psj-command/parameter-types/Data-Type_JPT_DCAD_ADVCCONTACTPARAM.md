## Description

A data type uses to control parameters of ADVC Contact.

## Attributes

### `iInitialState`

- An _Integer_ specifying initial state.
- The default value is -1. 

### `iBehavior`

- An _Integer_ specifying behavior.
- The default value is -1. 

### `dClearance`

- An _Integer_ specifying clearance.
- The default value is 0.1234. 

### `iAdjust2Clearance`

- An _Integer_ specifying adjust to clearance.
- The default value is -1 

### `iAdjust2Interference`

- An _Integer_ adjust to interference.
- The default value is -1. 

### `iAutoShrink`

- An _Integer_ specifying auto shrink.
- The default value is -1. 

### `iAdvAdjust`

- An _Integer_ specifying adjust.
- The default value is -1.

### `iEstimationImpactTime`

- An _Integer_ specifying estimation impact time.
- The default value is -1. 

### `iFormula`

- An _Integer_ specifying formula.
- The default value is -1.

### `iConstraintType`

- An _Integer_ specifying constraint type.
- The default value is -1.

### `iTyingType`

- An _Integer_ specifying typing type.
- The default value is -1.
