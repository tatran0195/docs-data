## Description

A data type used to control all parameters of Eribeam.

## Attributes

### `iElemId`
- An _Integer_ specifying element ID.
- The default value is _DFLT_INT_.

### `iPropId`

- An _Integer_ specifying property ID.
- The default value is _DFLT_INT_.

### `iEndA`
- An _Integer_ specifying node ID for endpoint A.
- The default value is _DFLT_INT_.

### `iEndB`

- An _Integer_ specifying node ID for endpoint B.
- The default value is _DFLT_INT_.

### `dlOrientVec`

- A _List of Double_ specifying orientation vector.
- The default value is _[DFLT_DBL, DFLT_DBL, DFLT_DBL]_.

### `iOrientNodeId`

- A _Integer_ specifying ID of orientation node.
- The default value is _DFLT_INT_.

### `dlOffsetVecA`

- A _List of Double_ specifying offset vector for endpoint A.
- The default value is _[DFLT_DBL, DFLT_DBL, DFLT_DBL]_.
 
### `dlOffsetVecB`

- A _List of Double_ specifying offset vector for endpoint B.
- The default value is _[DFLT_DBL, DFLT_DBL, DFLT_DBL]_.
 
### `iPinA`

- A _Integer_ specifying the pin flag solution degree of freedom at the endpoint A.
- The default value is _DFLT_INT_.

### `iPinB`

- A _Integer_ specifying the pin flag solution degree of freedom at the endpoint B.
- The default value is _DFLT_INT_.

### `dWarp`

- A _Double_ specifying the warping.
- The default value is _DFLT_DBL_.
