## Description

A data type uses to control parameters of Eribeam Offset Vec Prop

## Attributes

### `iPropId`

- An _Integer_ specifying prop ID.
- The default value is 0.

### `iElemId`

- An _Integer_ specifying elem ID.
- The default value is 0.

### `dlOffsetVec`

- A _Double List_ specifying offset vec.
- The default value is 0.
