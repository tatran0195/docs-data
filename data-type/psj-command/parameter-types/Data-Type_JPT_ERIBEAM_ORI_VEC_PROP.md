## Description

A data type uses to control parameters of Eribeam Ori Vec Prop

## Attributes

### `iPropId`

- An _Integer_ specifying prop ID.
- The default value is 0.

### `iElemId`

- An _Integer_ specifying elem ID.
- The default value is 0.

### `dlOrientVec`

- A _List of Double_ specifying orient vec.
- The default value is 0.
