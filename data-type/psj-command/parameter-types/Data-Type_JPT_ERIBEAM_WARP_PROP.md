## Description

A data type uses to control parameters of Eribeam Warp Prop

## Attributes

### `iPropId`
- An _Integer_ specifying prop ID.
- The default value is 0.

### `iElemId`
- An _Integer_ specifying elem ID.
- The default value is 0.
