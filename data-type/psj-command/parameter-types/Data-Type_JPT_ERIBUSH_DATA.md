## Description

A data type used to control all parameters of Eribush.

## Attributes

### `iElemId`
- An _Integer_ specifying element ID.
- The default value is _DFLT_INT_.

### `iEndA`
- An _Integer_ specifying node ID for endpoint A.
- The default value is _DFLT_INT_.

### `iEndB`

- An _Integer_ specifying node ID for endpoint B.
- The default value is _DFLT_INT_.

### `dlOrientVec`

- A _List of Double_ specifying orientation vector.
- The default value is _[DFLT_DBL, DFLT_DBL, DFLT_DBL]_.

### `iNodeId`

- A _Integer_ specifying the ID of reference node.
- The default value is _DFLT_INT_.

### `iCId`

- A _Integer_ specifying the reference coordinate system.
- The default value is _DFLT_INT_.

### `dDamperLoc`

- A _List of Double_ specifying the position of the bush element as the distance from end A.
- The default value is _[DFLT_DBL, DFLT_DBL, DFLT_DBL]_.
  
### `iOcid`

- A _Integer_ specifying the offset coordinate system.
- The default value is _DFLT_INT_.

### `dlOrientVec`

- A _List of Double_ specifying  the offset vector for the bush element in direction cosines.
- The default value is _[DFLT_DBL, DFLT_DBL, DFLT_DBL]_.
