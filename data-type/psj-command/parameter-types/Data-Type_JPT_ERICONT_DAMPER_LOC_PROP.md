## Description

A data type uses to control parameters of Ericont Damper Loc Prop

## Attributes

### `iConnId`

- An _Integer_ specifying connection ID.
- The default value is 0.

### `iEndA`

- An _Integer_ specifying end a.
- The default value is 0.

### `iEndB`

- An _Integer_ specifying end b.
- The default value is 0.
