## Description

A data type uses to control parameters of Ericont Damper Offset Vec Prop

## Attributes

### `iConnId`

- An _Integer_ specifying connection ID.
- The default value is 0.

### `iEndA`

- An _Integer_ specifying end A.
- The default value is 0.

### `iEndB`

- An _Integer_ specifying end B.
- The default value is 0.

### `dAmperOffsetVecX`

- A _Double_ specifying amper offset vec x.
- The default value is 0.0.

### `dAmperOffsetVecY`

- A _Double_ specifying amper offset vec y.
- The default value is 0.0.
