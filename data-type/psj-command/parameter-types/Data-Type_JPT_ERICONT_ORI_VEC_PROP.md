## Description

A data type uses to control parameters of Ericont Ori Vec Prop

## Attributes

### `iConnId`

- An _Integer_ specifying connection ID.
- The default value is 0.

### `iEndA`

- An _Integer_ specifying end a.
- The default value is 0.

### `iEndB`

- An _Integer_ specifying end b.
- The default value is 0.

### `dOrientVecX`

- A _Double_ specifying orient vec x.
- The default value is 0.0.

### `dOrientVecY`

- A _Double_ specifying orient vec y.
- The default value is 0.0.
