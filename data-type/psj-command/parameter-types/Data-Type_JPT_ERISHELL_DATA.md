## Description

A data type used to control all parameters of Eribeam.

## Attributes

### `iElemId`
- An _Integer_ specifying element ID.
- The default value is _DFLT_INT_.

### `iPropId`

- An _Integer_ specifying property ID.
- The default value is _DFLT_INT_.

### `dTheta`
- An _Double_ specifying orientation angle for material property.
- The default value is _DFLT_DBL_.

### `iCS`

- An _Integer_ specifying orientation coordinate system for material property.
- The default value is _DFLT_INT_.

### `dZoffs`

- A _Double_ specifying Z offset value.
- The default value is _DFLT_DBL_.
