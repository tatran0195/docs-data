## Description

A data type uses to control parameters of Linear Static data.

## Attributes

### `strPartName`

- An _Integer_ specifying the part name.
- The default value is "".

### `strPropName`

- A _String_ specifying the property name.
- The default value is "".

### `strMatName`

- A _String_ specifying the material name.
- The default value is "".

### `dValue`

- A _String_ specifying the the standard value.
- The default value is 0.0.
