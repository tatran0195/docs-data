## Description

A data type uses to control parameters of Grid Mesh

## Attributes

### `crlFace`

- A _List of Cursor_ specifying face.
- The default value is [].

### `crlCorner`

- A _List of Cursor_ specifying corner.
- The default value is [].

### `ilMeshCount`

- A _List of Integer_ specifying mesh count.
- The default value is [].

### `iShape`

- An _Integer_ specifying shape.
- The default value is -1.

### `iElemtype`

- An _Integer_ specifying element type.
- The default value is 3.

### `bSwap`

- A _Boolean_ enable/disable swap.
- The default value is False.

### `bLocalSetting`

- A _Boolean_ enable/disable local setting.
- The default value is False.

### `bOptimize`

- A _Boolean_ enable/disable optimize.
- The default value is False.
