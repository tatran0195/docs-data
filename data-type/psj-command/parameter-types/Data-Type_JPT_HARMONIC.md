## Description

A data type uses to control parameters of Harmonic

## Attributes

### `dHarmonicStartFreq`

- A _Double_ specifying harmonic start frequency.
- The default value is 0.0.

### `dHarmonicEndFreq`

- A _Double_ specifying harmonic end frequency.
- The default value is 1.0.

### `iSubSteps`

- An _Integer_ specifying sub steps.
- The default value is 0.

### `dAlphaD`

- A _Double_ specifying alpha d.
- The default value is 0.0.

### `dBetaD`

- A _Double_ specifying beta d.
- The default value is 0.0.

### `dDmprat`

- A _Double_ specifying dmprat.
- The default value is 0.0.

### `bHarmonicOutputDisplacements`

- A _Boolean_ enable/disable harmonic output displacements.
- The default value is False.

### `bHarmonicOutputStress`

- A _Boolean_ enable/disable harmonic output stress.
- The default value is False.
