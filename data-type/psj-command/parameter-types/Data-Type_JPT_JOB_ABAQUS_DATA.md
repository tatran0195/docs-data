## Description

A data type uses to control parameters of Job Abaqus Data

## Attributes

### `bRBE2toMPC`

- A _Boolean_ enable/disable r b e2to m p c.
- The default value is False.

### `bRenameProcess`

- A _Boolean_ enable/disable rename process.
- The default value is False.

### `iCodeType`

- An _Integer_ specifying code type.
- The default value is 0.

### `iSurfDefType`

- An _Integer_ specifying surf def type.
- The default value is 0.

### `iUnit`

- An _Integer_ specifying unit.
- The default value is 0.

### `iWriteType`

- An _Integer_ specifying write type.
- The default value is 0.

### `strDescription`

- A _String_ specifying description.
- The default value is "".

### `strlUserText`

- A _List of String_ specifying user text.
- The default value is ].

### `bExptNdEleGroups`

- A _Boolean_ enable/disable expt nd ele groups.
- The default value is False.

### `bDeleteFloatingNodes`

- A _Boolean_ enable/disable delete floating nodes.
- The default value is False.

### `bExptFaceElemGroups2Surface`

- A _Boolean_ enable/disable expt face elem groups2 surface.
- The default value is False.

### `bLoadCase`

- A _Boolean_ enable/disable load case.
- The default value is False.

### `bAutoAssignDummyProperty`

- A _Boolean_ enable/disable auto assign dummy property.
- The default value is True.

### `crDummyMat`

- A _Cursor_ specifying dummy material.
- The default value is None.
