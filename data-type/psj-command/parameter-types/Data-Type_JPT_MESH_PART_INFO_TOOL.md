## Description

A data type uses to control parameters of Mesh Part Info Tool

## Attributes

### `strName`

- A _String_ specifying part name.
- The default value is "".

### `iPartID`

- An _Integer_ specifying part i d.
- The default value is 0.

### `strPropName`

- A _String_ specifying prop name.
- The default value is "".

### `iPropId`

- An _Integer_ specifying prop ID.
- The default value is 0.

### `iPropType`

- An _Integer_ specifying prop type.
- The default value is 0.

### `strMatName`

- A _String_ specifying material name.
- The default value is "".

### `iMatID`

- An _Integer_ specifying material i d.
- The default value is 0.

### `strMatType`

- A _String_ specifying material type.
- The default value is "".

### `iPartNode`

- An _Integer_ specifying part node.
- The default value is 0.

### `iNpartnodeWithprop`

- An _Integer_ specifying npartnode withprop.
- The default value is 0.

### `ilPartElement`

- A _List of Integer_ specifying part element.
- The default value is [].
