## Description

A data type uses to control parameters of Modal

## Attributes

### `iModeMethod`

- An _Integer_ specifying mode method.
- The default value is 0.

### `iExtractNum`

- An _Integer_ specifying extract number.
- The default value is 1.

### `bExpandShape`

- A _Boolean_ enable/disable expand shape.
- The default value is True.

### `iExpandNum`

- An _Integer_ specifying expand number.
- The default value is 0.

### `bUseApprox`

- A _Boolean_ enable/disable use approx.
- The default value is False.

### `bInclPrestressEffect`

- A _Boolean_ enable/disable incl prestress effect.
- The default value is False.

### `bModalMemorySave`

- A _Boolean_ enable/disable modal memory save.
- The default value is False.

### `bResidualVector`

- A _Boolean_ enable/disable residual vector.
- The default value is False.

### `bModalOutputDisplacements`

- A _Boolean_ enable/disable modal output displacements.
- The default value is False.

### `bModalOutputStrain`

- A _Boolean_ enable/disable modal output strain.
- The default value is False.
