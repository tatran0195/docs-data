## Description

A data type uses to control parameters of Nastran Eigen

## Attributes

### `dStartFreq`

- A _Double_ specifying the starting frequency value.
- The default value is DFLT_DBL.

### `dEndFreq`

- A _Double_ specifying the ending frequency value.
- The default value is DFLT_DBL.

### `iNoOfModes`

- A _Double_ specifying the number of modes.
- The default value is DFLT_DBL.
