## Description

A data type uses to control parameters of subcase map.

## Attributes

### `iResultSet`

- An _Integer_ specifying the result set.
- The default value is 0.

### `listSubcaseIDs`

- A _List of Integer_ specifying the IDs of the specified subcases.
- The default value is "".
