## Description

This is an instance of a PostDataOp class, represents Post Data Option information in Jupiter.

## Properties

| Attribute      | Description                                                                                              |
| -------------- | -------------------------------------------------------------------------------------------------------- |
| loc            | [Location](../enumeration-types/post-result-data-loc-types) option information.                          |
| cnv            | [Conversion](../enumeration-types/post-result-data-cnv-types) option information.                        |
| cont           | [Continuous Conversion](../enumeration-types/post-result-data-cont-types) option information.            |
| coord          | [Coordinate](../enumeration-types/post-result-data-coord-types) option information.                      |
| load1D         | [Load 1D](../enumeration-types/post-result-data-load1d-types) option information.                        |
| load2D         | [Load 2D](../enumeration-types/post-result-data-load2d-types) option information.                        |
| complex        | [Complex](../enumeration-types/post-result-data-complex-types) option information.                       |
| phaseAngle     | Phase angle value.                                                                                       |
| userCoordSysId | ID of User-defined Coordinate System. ID equals `0` if none of user-defined Coordinate System specified. |
