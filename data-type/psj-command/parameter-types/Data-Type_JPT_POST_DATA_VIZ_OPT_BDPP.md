## Description

A data type is used to set up the result settings for displaying the FFT3D.

## Attributes

### `dLayerPointSize`

- A _Double_ specifying the size of displacement points on each layer.
- The default value is 1.0.

### `dLineWidth`

- A _Double_ specifying the width of the cylinder shape line.
- The default value is 1.0.

### `dPointSize`

- A _Double_ specifying the size of dots.
- The default value is 1.0.

### `dDotDistance`

- A _Double_ specifying the distance between dot points.
- The default value is 0.02.
