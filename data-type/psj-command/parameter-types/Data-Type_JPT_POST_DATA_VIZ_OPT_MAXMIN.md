## Description

A data type is used to set up the result settings for displaying Max/Min value.

## Attributes

### `iGroupMethod`

- An _Integer_ specifying the the group method.
    - 0: For All 
    - 1: For Each 
- The default value is 0.

### `iMaxColor`

- An _Integer_ specifying the maximum color.
- The default value is 255.

### `iMinColor`

- An _Integer_ specifying the minimum color.
- The default value is 16711680.

- An _Integer_ specifying the .
- The default value is 65535.
