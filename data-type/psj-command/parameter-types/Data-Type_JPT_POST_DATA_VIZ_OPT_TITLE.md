## Description

A data type is used to set up the result settings for displaying the title.

## Attributes

### `bResultName`

- A _Boolean_ specifying whether to display result name in the main window.
- The default value is _True_.

### `bConvection`

- A _Boolean_ specifying whether to display convection in the main window.
- The default value is _True_.

### `bUnit`

- A _Boolean_ specifying whether to display unit in the main window.
- The default value is _True_.

### `bCoordinate`

- A _Boolean_ specifying whether to display coordinate in the main window.
- The default value is _True_.

### `bTotalMaxMin`

- A _Boolean_ specifying whether to display total max/min in the main window.
- The default value is _True_.

### `bFileName`

- A _Boolean_ specifying whether to display file name in the main window.
- The default value is _True_.

### `bLocation`

- A _Boolean_ specifying whether to display location in the main window.
- The default value is _True_.

### `bVisibleMaxMin`

- A _Boolean_ specifying whether to display visible max/min in the main window.
- The default value is _True_.

### `bAnimation`

- A _Boolean_ specifying whether to display animation in the main window.
- The default value is _True_.

### `bDeformRatio`

- A _Boolean_ specifying whether to display deformation ratio in the main window.
- The default value is _True_.

### `bDetail`

- A _Boolean_ specifying whether to display detail in the main window.
- The default value is _True_.

### `iResultTitleSize`

- An _Integer_ specifying the index option of the result title's font size.
- The default value is 6.

### `iResultDescriptionSize`

- An _Integer_ specifying the index option of the result description's font size.
- The default value is 5.

### `iResultTitleColor`

- An _Integer_ specifying the font color of the result title.
- The default value is 16777215.

### `bResultTitleBold`

- A _Boolean_ specifying whether to display result title in bold mode.
- The default value is _False_.

### `iResultDescriptionColor`

- An _Integer_ specifying the font color of the result description.
- The default value is 16777215.

### `bResultDescriptionBold`

- A _Boolean_ specifying whether to display result description in bold mode.
- The default value is _False_.

### `bUseBackground`

- A _Boolean_ specifying whether to use background color.
- The default value is _False_.

### `iBackgroundFillColor`

- An _Integer_ specifying the background fill color.
- The default value is 16777215.

### `iBackgroundBorderColor`

- An _Integer_ specifying the background border color.
- The default value is 16777215.

### `dTransparency`

- An _Integer_ specifying the background transparency.
- The default value is 0.5.

### `bAdjustTitlePos`

- A _Boolean_ specifying whether to adjust title position for image capture.
- The default value is _True_.

### `bDeformCoordinate`

- A _Boolean_ specifying whether to display deformation coordinate in the main window.
- The default value is _False_.

### `bStep`

- A _Boolean_ specifying whether to display step information in the main window.
- The default value is _True_.

### `bIncrement`

- A _Boolean_ specifying whether to display increment information in the main window.
- The default value is _True_.

### `bResult`

- A _Boolean_ specifying whether to display result information in the main window.
- The default value is _True_.

### `bComponent`

- A _Boolean_ specifying whether to display component information in the main window.
- The default value is _True_.
