## Description

A data type is used to set up the result settings for displaying the opacity.

## Attributes

### `iTransparencyOpt`

- An _Integer_ specifying the transparency option.
- The default value is 0.

### `dTransparency`

- A _Double_ specifying the transparency value.
- The default value is 0.0.

### `dOpaqueRangeMin`

- A _Double_ specifying the opaque range min.
- The default value is 0.0.

### `dOpaqueRangeMax`

- A _Double_ specifying the opaque range max.
- The default value is 0.0.
