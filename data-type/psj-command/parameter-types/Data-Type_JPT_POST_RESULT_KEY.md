## Description

A data type uses to identify the selected result.

## Attributes

### `iAnalysisType`

- An _Integer_ specifying the analysis type.
- The default value is 0.

### `iAnalysisID`

- A _String_ specifying the analysis ID.
- The default value is 0.

### `iResultSet`

- A _String_ specifying the result set.
- The default value is 0.

### `iTimeStep`

- A _String_ specifying the time step.
- The default value is 0.

### `iResultType`

- A _String_ specifying the category type.
- The default value is 0.

### `strResultName`

- A _String_ specifying the result name.
- The default value is "".

### `strResultCompName`

- A _String_ specifying the result component name.
- The default value is "".

### `iResultPos`

- A _String_ specifying the result position.
- The default value is 0.
