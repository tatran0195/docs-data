## Description

A data type uses to control parameters of Post Step Item.

## Attributes

### `postTimeStepInfo`

- A _Class [POST_TIME_STEP_INFO](POST_TIME_STEP_INFO)_ specifying the attributes of Post Time Step Information.
- The default value is POST_TIME_STEP_INFO().

### `iAnalysisType`

- An _Integer_ specifying the Analysis type.
- The default value is 0.

### `crPostJob`

- A _Cursor_ specifying the Post Job.
- The default value is None.

### `iResultSet`

- An _Integer_ specifying the Result Set.
- The default value is 0.

### `iTimeStep`

- An _Integer_ specifying the Time Step.
- The default value is 0.
