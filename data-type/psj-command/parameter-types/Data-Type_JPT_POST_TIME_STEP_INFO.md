## Description

A data type uses to define the information of post time step.

## Attributes

### `iMode`

- An _Integer_ specifying the using mode.
- The default value is DFLT_INT.

### `dTime`

- A _Double_ specifying the value of time.
- The default value is FLT_MAX.

### `dFrequency`

- A _Double_ specifying the frequency value.
- The default value is FLT_MAX.

### `iBaseAnalysisType`

- An _Integer_ specifying the base analysis type.
- The default value is 0.

### `strResultSetName`

- A _String_ specifying the result set name.
- The default value is "".
