## Description

A data type uses to control parameters of pretension load.

## Attributes

### `iSolverType`
- An _Int_ specifying Solver type.
  - 0: All
  - 1: Abaqus 
- The default value is 0.

### `dValue`
- A _Double_ specifying pretension load.
- The default value is 0.
  
### `iUnit`
- An _Int_ specifying input unit setting.
- The default value is 0.
  
### `bBoltFixLength`
- A _Bool_ specifying whether create bolt fix length.
- The default value is _False_.
