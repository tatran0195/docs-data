## Description

A data type uses to control parameters of Rbar Connection

## Attributes

### `strName`

- A _String_ specifying name.
- The default value is "".

### `iEType`

- An _Integer_ specifying e type.
- The default value is 7.

### `iMethod`

- An _Integer_ specifying method.
- The default value is 1.

### `crCoordSys`

- A _Cursor_ specifying coord sys.
- The default value is None.
