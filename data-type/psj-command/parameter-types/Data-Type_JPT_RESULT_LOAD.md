## Description

A data type uses to control parameters of Result Load.

## Attributes

### `iVrType`

- An _Integer_ specifying the virtual type.
- The default value is 0.

### `strResultName`

- A _String_ specifying the result name.
- The default value is "".

### `strLoadName`

- A _String_ specifying the load name.
- The default value is "".
