## Description

A data type uses to identify the selected result.

## Attributes

### `crReferencePostJob`

- A _Cursor_ specifying the reference post job.
- The default value is None.

### `strName`

- A _String_ specifying the variable name.
- The default value is "".

### `iAnalysisType`

- An _Integer_ specifying the analysis type.
- The default value is 0.

### `iResultSet`

- An _Integer_ specifying the result step.
- The default value is 0.

### `iTimeStep`

- An _Integer_ specifying the time step.
- The default value is 0.

### `iResultType`

- An _Integer_ specifying the result type.
- The default value is 0.

### `iResultPos`

- An _Integer_ specifying the result position.
- The default value is 0.
