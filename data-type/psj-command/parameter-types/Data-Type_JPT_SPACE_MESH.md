## Description

A data type uses to control parameters of Space Mesh

## Attributes

### `iBodyType`

- An _Integer_ specifying body type.
- The default value is 0.

### `bEnclosed`

- A _Boolean_ enable/disable enclosed.
- The default value is False.

### `vOrigin`

- A _V_ORIGIN_ specifying origin.
- The default value is 0.0.

### `vCenterPt`

- A _V_CENTER_PT_ specifying center pt.
- The default value is 0.0.

### `vLength`

- A _V_LENGTH_ specifying length.
- The default value is 0.01.

### `vAxisPt1`

- A _V_AXIS_PT1_ specifying axis pt1.
- The default value is 0.0.

### `vAxisPt2`

- A _V_AXIS_PT2_ specifying axis pt2.
- The default value is 0.0.

### `dMaxElemSize`

- A _Double_ specifying maximum elem size.
- The default value is DFLT_DBL.
