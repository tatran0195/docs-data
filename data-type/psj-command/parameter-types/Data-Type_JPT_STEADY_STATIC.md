## Description

A data type uses to control parameters of Steady Static

## Attributes

### `bSteadyStaticMemorySave`

- A _Boolean_ enable/disable steady static memory save.
- The default value is False.

### `bSteadyStaticOutputHeatFlux`

- A _Boolean_ enable/disable steady static output heat flux.
- The default value is False.

### `bSteadyStaticOutputTemperature`

- A _Boolean_ enable/disable steady static output temperature.
- The default value is False.

### `bPivotsCheck`

- A _Boolean_ enable/disable pivots check.
- The default value is True.

### `bSteadyStaticSinglePrecision`

- A _Boolean_ enable/disable steady static single precision.
- The default value is False.

### `dSteadyStaticMultiplier`

- A _Double_ specifying steady static multiplier.
- The default value is 0.0.

### `dSteadyStaticTemperatureDifference`

- A _Double_ specifying steady static temperature difference.
- The default value is 0.0.

### `dSteadyStaticToleranceLevel`

- A _Double_ specifying steady static tolerance level.
- The default value is 0.0.

### `iAdaptiveDescent`

- An _Integer_ specifying adaptive descent.
- The default value is 0.

### `iSteadyStaticEquationSolver`

- An _Integer_ specifying steady static equation solver.
- The default value is 0.
