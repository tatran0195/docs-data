## Description

A data type uses to control parameters of Surface Mesh

## Attributes

### `dAvgElemSize`

- A _Double_ specifying avg elem size.
- The default value is 0.005.

### `dMaxElemSize`

- A _Double_ specifying maximum elem size.
- The default value is 0.01.

### `dMinElemSize`

- A _Double_ specifying minimum elem size.
- The default value is 0.001.

### `dReductionFactor`

- A _Double_ specifying reduction factor.
- The default value is 1.0.

### `dGeomAngle`

- A _Double_ specifying geom angle.
- The default value is 0.785398.

### `dGeomMinSize`

- A _Double_ specifying geom minimum size.
- The default value is 0.001.

### `dGradingFactor`

- A _Double_ specifying grading factor.
- The default value is 1.250000.

### `dMinStretchVal`

- A _Double_ specifying minimum stretch val.
- The default value is 0.100000.

### `dGeomEdgeDev`

- A _Double_ specifying geom edge dev.
- The default value is 0.100000.

### `dGeomQualRatio`

- A _Double_ specifying geom qual ratio.
- The default value is 0.700000.

### `dGeomCntRatio`

- A _Double_ specifying geom cnt ratio.
- The default value is 0.500000.

### `iPerformanceMode`

- An _Integer_ specifying performance mode.
- The default value is 0.

### `iOptLevel`

- An _Integer_ specifying opt level.
- The default value is 3.

### `bAutoMergeEdges`

- A _Boolean_ enable/disable auto merge edges.
- The default value is False.

### `bAutoMergeFaces`

- A _Boolean_ enable/disable auto merge faces.
- The default value is False.

### `dAutoMeshPatternMinElemAngle`

- A _Double_ specifying auto mesh pattern minimum elem angle.
- The default value is 0.

### `dAutoMergeTinyFacesAngle`

- A _Double_ specifying auto merge tiny faces angle.
- The default value is 0.523598756.

### `bOutputQuadMesh`

- A _Boolean_ enable/disable output quad mesh.
- The default value is False.

### `bPureQuad`

- A _Boolean_ enable/disable pure quad.
- The default value is False.

### `bBadInputModel`

- A _Boolean_ enable/disable bad input model.
- The default value is False.

### `bCloseGaps`

- A _Boolean_ enable/disable close gaps.
- The default value is False.

### `bLocalRemesh`

- A _Boolean_ enable/disable local remesh.
- The default value is False.

### `bGeomApprox`

- A _Boolean_ enable/disable geom approx.
- The default value is False.

### `bCurvatureControl`

- A _Boolean_ enable/disable curvature control.
- The default value is False.

### `bDeleteCircChamfer`

- A _Boolean_ enable/disable delete circ chamfer.
- The default value is False.

### `bTinyCylinderMesh`

- A _Boolean_ enable/disable tiny cylinder mesh.
- The default value is False.

### `iNextEntityOffsetId`

- An _Integer_ specifying next entity offset ID.
- The default value is 10000000.

### `iNextElemOffsetId`

- An _Integer_ specifying next elem offset ID.
- The default value is 0.

### `iNextNodeOffsetId`

- An _Integer_ specifying next node offset ID.
- The default value is 0.
