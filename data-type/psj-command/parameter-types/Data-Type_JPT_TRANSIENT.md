## Description

A data type uses to control parameters of Transient

## Attributes

### `bTransientEffect`

- A _Boolean_ enable/disable transient effect.
- The default value is True.

### `iLoadingType`

- An _Integer_ specifying loading type.
- The default value is 0.

### `dMassMatrixMultiplier`

- A _Double_ specifying mass matrix multiplier.
- The default value is 0.0.

### `dStiffMatrixMultiplier`

- A _Double_ specifying stiff matrix multiplier.
- The default value is 0.0.

### `bMidStep`

- A _Boolean_ enable/disable mid step.
- The default value is False.

### `dToleranceBisection`

- A _Double_ specifying tolerance bisection.
- The default value is 0.0.

### `dToleranceTimeStep`

- A _Double_ specifying tolerance time step.
- The default value is 0.0.

### `iTimeInterAlgorithm`

- An _Integer_ specifying time inter algorithm.
- The default value is 0.

### `iTimeInter`

- An _Integer_ specifying time inter.
- The default value is 0.

### `dGamma`

- A _Double_ specifying gamma.
- The default value is 0.005.

### `dAlpha`

- A _Double_ specifying alpha.
- The default value is 0.25250625.

### `dDelta`

- A _Double_ specifying delta.
- The default value is 0.505.

### `dAlphaF`

- A _Double_ specifying alpha f.
- The default value is 0.005.

### `dAlphaM`

- A _Double_ specifying alpha m.
- The default value is 0.0.

### `bTransientOutputTemperature`

- A _Boolean_ enable/disable transient output temperature.
- The default value is False.
