---
title: MessageBox Button Types
id: msgbox-button-types
---

This is an enumeration type represents MessageBox Button Types.

| MessageBox Button Types              | Description                                                      |
| ------------------------------------ | ---------------------------------------------------------------- |
| `msgbox_buttons.ok`                  | Display message box with Ok button                               |
| `msgbox_buttons.ok_cancel`           | Display message box with Ok and Cancel button                    |
| `msgbox_buttons.yes_no`              | Display message box with Yes and No button                       |
| `msgbox_buttons.retry_cancel`        | Display message box with Retry and Cancel button                 |
| `msgbox_buttons.yes_no_cancel`       | Display message box with Yes, No and Cancel button               |
| `msgbox_buttons.abort_retry_ignore`  | Display message box with Abort, Retry, and Ignore buttons        |
| `msgbox_buttons.cancel_try_continue` | Display message box with Cancel, Try Again, and Continue buttons |
