---
title: MessageBox Icon Types
id: msgbox-icon-types
---

This is an enumeration type represents MessageBox Icon Types.

| MessageBox Icon Types      | Description                               |
| -------------------------- | ----------------------------------------- |
| `msgbox_icon.default`      | Display message box with Information icon |
| `msgbox_icon.error`        | Display message box with Error icon       |
| `msgbox_icon.exclamation`  | Display message box with Exclamation icon |
| `msgbox_icon.question`     | Display message box with Question icon    |
| `msgbox_icon.asterisk`     | Display message box with Asterisk icon    |
| `msgbox_icon.user`         | Display message box with User icon        |
