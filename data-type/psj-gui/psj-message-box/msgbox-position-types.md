---
title: MessageBox Position Types
id: msgbox-position-types
---

This is an enumeration type represents MessageBox Position Types.

| MessageBox Position Types  | Description                          |
| -------------------------- | ------------------------------------ |
| `msgbox_position.first`    | Add button at first position         |
| `msgbox_position.second`   | Add button at second position        |
| `msgbox_position.third`    | Add button at third position         |
| `msgbox_position.last`     | Add button at last (fourth) position |
