---
title: Post Analysis Types
id: post-analysis-types
---

This is an enumeration type represents Post Analysis type.

| ID  | Post Analysis type                                          | Description                                                                                        |
| --- | ----------------------------------------------------------- | -------------------------------------------------------------------------------------------------- |
| 1   | `JPT.PostAnalysisType.POST_ANALYSIS_LINEAR_STATIC`          | Linear Static Analysis.                                                                            |
| 2   | `JPT.PostAnalysisType.POST_ANALYSIS_MODAL`                  | Normal Modes Static Analysis.                                                                      |
| 3   | `JPT.PostAnalysisType.POST_ANALYSIS_COMPLEX_MODAL`          | Complex Modal Modes Analysis.                                                                      |
| 4   | `JPT.PostAnalysisType.POST_ANALYSIS_FREQUENCY`              | Frequency Response Analysis.                                                                       |
| 5   | `JPT.PostAnalysisType.POST_ANALYSIS_TRANSIENT`              | Transient Response Analysis.                                                                       |
| 6   | `JPT.PostAnalysisType.POST_ANALYSIS_NONLINEAR`              | Nonlinear Analysis (Static, Buckling, Transient Response, Steady State Conduction, Heat Transfer). |
| 7   | `JPT.PostAnalysisType.POST_ANALYSIS_BUCKLING`               | Linear Buckling Analysis.                                                                          |
| 8   | `JPT.PostAnalysisType.POST_ANALYSIS_STEADY_HEAT_CONDUCTION` | Steady Heat Conduction Analysis.                                                                   |
| 9   | `JPT.PostAnalysisType.POST_ANALYSIS_HEAT_TRANSFER`          | Heat Transfer Analysis.                                                                            |
| 10  | `JPT.PostAnalysisType.POST_ANALYSIS_SHAPE_OPTIMIZATION`     | Shape Optimization Analysis.                                                                       |
| 11  | `JPT.PostAnalysisType.POST_ANALYSIS_TOPOLOGY_OPTIMIZATION`  | Topology Optimization Analysis.                                                                    |
| 12  | `JPT.PostAnalysisType.POST_ANALYSIS_USER_DEFINED`           | User defined Analysis.                                                                             |
