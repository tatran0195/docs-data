---
title: Post Result Data Complex Types
id: post-result-data-complex-types
---

This is an enumeration type represents Post Result Data Complex type.

| ID  | Post Result Data Complex type                          | Description                                                    |
| --- | ------------------------------------------------------ | -------------------------------------------------------------- |
| 0   | `JPT.PostResultDataComplex.POST_COMPLEX_UNKNOWN`       | Undefined Complex.                                             |
| 1   | `JPT.PostResultDataComplex.POST_COMPLEX_REAL`          | Display the Real result.                                       |
| 2   | `JPT.PostResultDataComplex.POST_COMPLEX_IMG`           | Display the Imaginary result.                                  |
| 4   | `JPT.PostResultDataComplex.POST_COMPLEX_AMP`           | Display Magnitude result.                                      |
| 8   | `JPT.PostResultDataComplex.POST_COMPLEX_PHASE`         | Display Phase result.                                          |
| 16  | `JPT.PostResultDataComplex.POST_COMPLEX_PHASE_ANGLE`   | Display the result at the specified Phase angle.               |
| 32  | `JPT.PostResultDataComplex.POST_COMPLEX_AMP_AND_PHASE` | Display the result at the specified Magnitude and Phase angle. |
