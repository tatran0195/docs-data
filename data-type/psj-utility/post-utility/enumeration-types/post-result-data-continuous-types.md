title: Post Result Data Continuous Types
id: post-result-data-cont-types
---

This is an enumeration type represents Post Result Data Continuous Conversion type.

| ID  | Post Result Data Continuous Conversion type | Description                                                                                                     |
| --- | ------------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| 0   | `JPT.PostResultDataCont.POST_CONT_UNKNOWN`  | Undefined Continuous Conversion.                                                                                |
| 1   | `JPT.PostResultDataCont.POST_CONT_MAT`      | Continuous Conversion by Material (Discontinuous contour displays at material boundary).                        |
| 2   | `JPT.PostResultDataCont.POST_CONT_PROP`     | Continuous Conversion by Property (Discontinuous contour displays at element property characteristic boundary). |
| 4   | `JPT.PostResultDataCont.POST_CONT_BLK`      | Continuous Conversion by Part (Block).                                                                          |
| 8   | `JPT.PostResultDataCont.POST_CONT_ALL`      | Conversion by none of above (Average by all element node results).                                              |
