This is an enumeration type represents Post Result Data Conversion type.

| ID  | Post Result Data Conversion type              | Description                     |
| --- | --------------------------------------------- | ------------------------------- |
| 0   | `JPT.PostResultDataCnv.POST_CNV_UNKNOWN`      | Undefined Conversion.           |
| 1   | `JPT.PostResultDataCnv.POST_CNV_AVG`          | Average Conversion.             |
| 2   | `JPT.PostResultDataCnv.POST_CNV_MAX`          | Maximum Conversion.             |
| 4   | `JPT.PostResultDataCnv.POST_CNV_MIN`          | Minimum Conversion.             |
| 8   | `JPT.PostResultDataCnv.POST_CNV_COMP_AVG`     | Component Average Conversion.   |
| 16  | `JPT.PostResultDataCnv.POST_CNV_ALL`          | All Conversion.                 |
| 32  | `JPT.PostResultDataCnv.POST_CNV_MAX_MIN`      | Max-Min Conversion.             |
| 64  | `JPT.PostResultDataCnv.POST_CNV_AVG_MATERIAL` | Average by Material Conversion. |
