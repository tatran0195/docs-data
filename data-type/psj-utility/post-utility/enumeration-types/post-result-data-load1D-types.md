This is an enumeration type represents Post Result Data Load 1D type.

| ID  | Post Result Data Load 1D type                  | Description                                          |
| --- | ---------------------------------------------- | ---------------------------------------------------- |
| 0   | `JPT.PostResultDataLoad1D.POST_LOAD1D_UNKNOWN` | Undefined 1D Data.                                   |
| 1   | `JPT.PostResultDataLoad1D.POST_LOAD1D_PT_1`    | Output stress recovery points 1 of the beam element. |
| 2   | `JPT.PostResultDataLoad1D.POST_LOAD1D_PT_2`    | Output stress recovery points 2 of the beam element. |
| 4   | `JPT.PostResultDataLoad1D.POST_LOAD1D_PT_3`    | Output stress recovery points 3 of the beam element. |
| 8   | `JPT.PostResultDataLoad1D.POST_LOAD1D_PT_4`    | Output stress recovery points 4 of the beam element. |
| 16  | `JPT.PostResultDataLoad1D.POST_LOAD1D_MAX`     | Output maximum result of the beam element.           |
| 32  | `JPT.PostResultDataLoad1D.POST_LOAD1D_MIN`     | Output minimum result of the beam element.           |
