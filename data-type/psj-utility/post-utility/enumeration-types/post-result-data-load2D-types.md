title: Post Result Data Load 2D Types
id: post-result-data-load2d-types
---

This is an enumeration type represents Post Result Data Load 2D type.

| ID  | Post Result Data Load 2D type                  | Description                                                                                |
| --- | ---------------------------------------------- | ------------------------------------------------------------------------------------------ |
| 0   | `JPT.PostResultDataLoad2D.POST_LOAD2D_UNKNOWN` | Undefined 2D Data.                                                                         |
| 1   | `JPT.PostResultDataLoad2D.POST_LOAD2D_TOP`     | Display the shell element top result.                                                      |
| 2   | `JPT.PostResultDataLoad2D.POST_LOAD2D_BOTTOM`  | Display the shell element bottom result.                                                   |
| 32  | `JPT.PostResultDataLoad2D.POST_LOAD2D_BOTH`    | Display the results of the top and bottom face on the front and back of the shell element. |
| 64  | `JPT.PostResultDataLoad2D.POST_LOAD2D_AVERAGE` | Display the average value of top and bottom face of the shell element.                     |
