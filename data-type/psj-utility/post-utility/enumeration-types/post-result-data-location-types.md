---
title: Post Result Data Location Types
id: post-result-data-loc-types
---

This is an enumeration type represents Post Result Data Location type.

| ID  | Post Result Data Location type                        | Description                          |
| --- | ----------------------------------------------------- | ------------------------------------ |
| 0   | `JPT.PostResultDataLoc.POST_LOC_UNKNOWN`              | Undefined Location.                  |
| 1   | `JPT.PostResultDataLoc.POST_LOC_ON_NODE`              | Result on Node (averaged).           |
| 2   | `JPT.PostResultDataLoc.POST_LOC_ON_ELEMENT`           | Result on Element.                   |
| 4   | `JPT.PostResultDataLoc.POST_LOC_ON_ELEMENT_NODE`      | Result on Element Node (unaveraged). |
| 8   | `JPT.PostResultDataLoc.POST_LOC_ON_INTEGRATION_POINT` | Result on Element Integration Point. |
