## Description

This is an instance of a DPostAnalysis class, represents a post analysis (target analysis) inside Jupiter.

## Properties

| Attribute   | Description                                                     |
| ----------- | --------------------------------------------------------------- |
| name        | The target analysis name.<br />The return value is a string of name (str).                                   |
| id          | ID of the target analysis.<br />The return value is ID (int).                                           |
| type        | Type of the target analysis.<br />The return value is string of post analysis type (str).               |
| solutionType | The analysis solution type.<br /> The return value is a string of solution type (str).                 |
| load        | Number of load conditions of the target analysis.<br />The return value is a number of loads (int).     |
| loadCase    | Number of the loadcases of the target analysis.<br />The return value is a number of loadcases (int).   |
| response    | Number of the responses of the target analysis.<br />The return value is a number of responses (int).   |
