## Description

This is an instance of a PostTimeStepInfo class, represents Post Time, Post Step, Post Frequency information of various analyses in Jupiter.

## Properties

| Attribute | Description                                     |
| --------- | ----------------------------------------------- |
| mode      | Mode number of Modal mode analysis.             |
| time      | Time value of Transient Heat analysis.          |
| freq      | Frequency value of Frequency Response analysis. |
