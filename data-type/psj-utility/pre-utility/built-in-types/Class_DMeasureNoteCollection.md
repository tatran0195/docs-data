## Description

This is an instance of a MeasureNoteCollection class, represents a measure note collections inside Jupiter.

## Properties

| Attribute | Description                                                                                                                                                                             |
| --------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| isValid   | For checking whether the indicated object is available or not. <br />The return value is bool.                                                                                          |
| typeID    | Type ID of the measure note collection(ID = 279).<br />The return value is an [`Int notation of DItem Types`](../../../psj-command/DItem-types) (int).                                  |
| tableID   | Table ID of the measure note collection item (ID = 161). The return value is an [`Int notation of DTable Types`](../../../psj-utility/pre-utility/enumeration-types/dtable-types)(int). |
| id        | External ID of the measure note collection.<br />The return value is ID (int).                                                                                                          |
| name      | Name of the measure note collection.<br />The return value is a string.                                                                                                                 |
