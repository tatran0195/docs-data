## Description

This is an instance of a Document class, represents Document information in Jupiter.

## Properties

| Attribute | Description                                                                                                                                                                          |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| docName   | Name of Document.<br />The return value is Name of active Document (string).                                                                                                         |
| docID     | ID of Document.<br />The return value is ID of active Document (string).                                                                                                             |
| docType   | Type of Document.<br />The return value is `DocPre` if active Document is Pre Document,<br/> `DocPost` if Post Document and `DocNone` if none of `DocPre` or `DocPost` is specified. |
| docPath   | Path of Document.<br/>The return value is Path of active Document (string).<br/> If active Document does not have specific path, `Undefined` will be the return value                |
