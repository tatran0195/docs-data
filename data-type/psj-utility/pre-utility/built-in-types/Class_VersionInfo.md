## Description

This is a class of JTDB/JTH5 file version, represents version information of Jupiter.

## Properties

| Attribute | Description           |
| --------- | --------------------- |
| major     | Major version number. |
| minor     | Minor version number. |
| sub       | Subversion number.    |
| build     | Build number.         |
