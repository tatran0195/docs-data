---
title: Associate Types
id: associate-types
---

This is an enumeration type represents Associated Pick function.

| ID  | Associate type                    | Description                                                                                                                                                                                                      |
| --- | --------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 0   | `JPT.AssociateType.AS_BARBODY`    | Specifying the Main window -> Right click -> Associated Pick -> Bar Part function                                                                                                                                |
| 1   | `JPT.AssociateType.AS_BODY`       | Specifying the Main window -> Right click -> Associated Pick -> Part function                                                                                                                                    |
| 2   | `JPT.AssociateType.AS_FACE`       | Specifying the Main window -> Right click -> Associated Pick -> Face function                                                                                                                                    |
| 3   | `JPT.AssociateType.AS_EDGE`       | Specifying the Main window -> Right click -> Associated Pick -> Edge function                                                                                                                                    |
| 4   | `JPT.AssociateType.AS_ELEM`       | Specifying the Main window -> Right click -> Associated Pick -> Element function                                                                                                                                 |
| 5   | `JPT.AssociateType.AS_ELEM1D`     | Specifying the Main window -> Right click -> Associated Pick -> 1D Element function                                                                                                                              |
| 6   | `JPT.AssociateType.AS_SOLIDELEM`  | Specifying the Main window -> Right click -> Associated Pick -> Solid Element function                                                                                                                           |
| 7   | `JPT.AssociateType.AS_NODE`       | Specifying the Main window -> Right click -> Associated Pick -> All Node function                                                                                                                                |
| 8   | `JPT.AssociateType.AS_CONNECTION` | Specifying the Main window -> Right click -> Associated Pick -> Connection -> Connection type (Such as RBE2, RBE3, etc.) function.<br />This function will get all the existing connections on the current model |
| 9   | `JPT.AssociateType.AS_MASS`       | Specifying the Main window -> Right click -> Associated Pick -> Connection -> Mass Element function.<br />This function will get all the existing mass elements on the current model                             |
| 10  | `JPT.AssociateType.AS_CONTACT`    | Specifying the Main window -> Right click -> Associated Pick -> Connection -> Contact function.<br />This function will get all the existing contacts on the current model                                       |
| 11  | `JPT.AssociateType.AS_GROUP`      | Specifying the Main window -> Right click -> Associated Pick -> Face/Element Group function                                                                                                                      |
