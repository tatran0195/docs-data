---
title: Message Console Types
id: msg-console-types
---

This is an enumeration type represents message console type.

| ID  | MsgBox type                               | Description                  |
| --- | ----------------------------------------- | ---------------------------- |
| 1   | `JPT.MsgConsoleType.MSG_EXECUTION`        | Execution message            |
| 2   | `JPT.MsgConsoleType.MSG_RETURN`           | Return message               |
| 3   | `JPT.MsgConsoleType.MSG_EXECUTION_RETURN` | Execution and return message |
| 4   | `JPT.MsgConsoleType.MSG_USER`             | User message (print/debug)   |
| 5   | `JPT.MsgConsoleType.MSG_ALL`              | All messages                 |
