---
title: Message Box Option Types
id: msgbox-option-types
---

This is an enumeration type represents message box option type.

| ID  | MsgBox Option type                      | Description             |
| --- | --------------------------------------- | ----------------------- |
| 1   | `JPT.MsgBoxOptionType.MB_OPTION_OK`     | Default value is OK     |
| 2   | `JPT.MsgBoxOptionType.MB_OPTION_CANCEL` | Default value is CANCEL |
| 6   | `JPT.MsgBoxOptionType.MB_OPTION_YES`    | Default value is YES    |
| 7   | `JPT.MsgBoxOptionType.MB_OPTION_NO`     | Default value is NO     |
