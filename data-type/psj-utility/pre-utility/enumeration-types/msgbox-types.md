---
title: Message Box Types
id: msgbox-types
---

This is an enumeration type represents message box type.

| ID  | MsgBox type                                 | Description                                            |
| --- | ------------------------------------------- | ------------------------------------------------------ |
| 0   | `JPT.MsgBoxType.MB_WARNING_YESNOCANCEL`     | Warning message box with Yes, No and Cancel button     |
| 1   | `JPT.MsgBoxType.MB_WARNING_YESNO`           | Warning message box with Yes and No button             |
| 2   | `JPT.MsgBoxType.MB_WARNING_OK`              | Warning message box with OK button                     |
| 3   | `JPT.MsgBoxType.MB_WARNING_OKCANCEL`        | Warning message box with OK and Cancel button          |
| 4   | `JPT.MsgBoxType.MB_INFORMATION_YESNOCANCEL` | Information message box with Yes, No and Cancel button |
| 5   | `JPT.MsgBoxType.MB_INFORMATION_YESNO`       | Information message box with Yes and No button         |
| 6   | `JPT.MsgBoxType.MB_INFORMATION_OK`          | Information message box with OK button                 |
| 7   | `JPT.MsgBoxType.MB_INFORMATION_OKCANCEL`    | Information message box with OK and Cancel button      |
| 8   | `JPT.MsgBoxType.MB_CRITICAL_OK`             | Critical message box with OK button                    |
