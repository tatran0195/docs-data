---
title: Path Types
id: path-types
---

This is an enumeration supporting get the full link to the specified path type.

| ID  | Path type                              | Description                                                                              |
| --- | -------------------------------------- | ---------------------------------------------------------------------------------------- |
| 0   | `JPT.PathType.PROGRAM_PATH`            | The full path to the current Jupiter installation folder                                 |
| 1   | `JPT.PathType.TEMP_PATH`               | The full path to the temp folder of Jupiter                                              |
| 2   | `JPT.PathType.APPDATA_PATH`            | The full path to the setting folder of Jupiter                                           |
| 3   | `JPT.PathType.DOC_PATH`                | The full path to the current Jupiter model                                               |
| 4   | `JPT.PathType.MATERIAL_LIB_FILE_PATH`  | The full path to the default Jupiter material library folder                             |
| 5   | `JPT.PathType.MATERIAL_USER_FILE_PATH` | The full path to the user-defined Jupiter material library folder                        |
| 6   | `JPT.PathType.DATABASE_PATH`           | The full path to the data base folder of Jupiter                                         |
| 7   | `JPT.PathType.SUNSHINE_PATH`           | The full path to the Sunshine folder of Jupiter                                          |
| 8   | `JPT.PathType.IDE_DATA_PATH`           | The full path to the IDE information (\*.dat file) of Jupiter                            |
| 9   | `JPT.PathType.FILE_PATH`               | The full path to the file from which the module was loaded, if it was loaded from a file |
| 10   | `JPT.PathType.PSJ_COMMAND_PATH`               | The full path to the user-defined PSJ Command folder |
